# <a name="azure-database-for-postgresql-documentation"></a>Документация по Базе данных Azure для PostgreSQL

База данных Azure для PostgreSQL — это служба реляционных баз данных на основе СУБД PostgreSQL с открытым исходным кодом. Эта полностью управляемая база данных как услуга может обслуживать критически важные рабочие нагрузки с прогнозируемой производительностью и безопасностью, а также высоким уровнем доступности и динамическим масштабированием.  Разрабатывайте приложения на основе Базы данных Azure для PostgreSQL, используя привычные средства с открытым исходным кодом и платформу.  Узнайте, как использовать Базу данных Azure для PostgreSQL, ознакомившись с нашими руководствами по началу работы, примерами и учебными материалами.

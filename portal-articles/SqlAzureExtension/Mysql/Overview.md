# <a name="azure-database-for-mysql-documentation"></a>Документация по базе данных Azure для MySQL

База данных Azure для MySQL — это служба реляционной базы данных на основе СУБД MySQL Server с открытым исходным кодом.  Эта полностью управляемая база данных как услуга может обслуживать критически важные рабочие нагрузки с прогнозируемой производительностью и динамическим масштабированием. Разрабатывайте приложения на основе базы данных Azure для MySQL, используя привычные средства с открытым исходным кодом и платформу. Узнайте, как использовать базу данных Azure для MySQL, ознакомившись с нашими шаблонами быстрого запуска, примерами и руководствами.

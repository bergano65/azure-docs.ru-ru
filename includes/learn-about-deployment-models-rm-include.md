> [!NOTE]
> В Azure доступны две модели развертывания, которые можно использовать для создания и работы с ресурсами: [развертывание с помощью Azure Resource Manager и классическая модель развертывания](../articles/azure-resource-manager/resource-manager-deployment-model.md). В этой статье рассматривается использование модели развертывания Resource Manager. Мы рекомендуем модели развертывания Resource Manager для новых развертываний, а не классической модели развертывания.

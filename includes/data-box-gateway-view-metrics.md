Кроме того, вы можете просматривать метрики для мониторинга производительности устройства и при необходимости устранять с их помощью неполадки.

Чтобы создать схему по выбранным метрикам устройства, выполните следующие действия на портале Azure:

1. На портале Azure откройте раздел **Мониторинг > Метрики** для нужного ресурса и щелкните **Добавить метрику**.

    ![Добавление метрики](media/data-box-gateway-view-metrics/view-metrics-add-metric.png)

2. Ресурс указывается автоматически.  

    ![Текущий ресурс](media/data-box-gateway-view-metrics/view-metrics-current-resource.png)

    Чтобы указать другой ресурс, выберите его. В колонке **Выберите ресурс** выберите подписку, группу ресурсов, тип ресурса и конкретный ресурс, для которого вам нужно отобразить метрики, затем щелкните **Применить**.

    ![Выбор другого ресурса](media/data-box-gateway-view-metrics/view-metrics-choose-another-resource.png)

3. В раскрывающемся списке выберите метрику для мониторинга устройства. Полный список этих метрик см. в разделе [Метрики вашего устройства](#metrics-on-your-device).

4. При выборе метрики из раскрывающегося списка вы можете дополнительно определить функцию статистической обработки. Статистическое выражение вычисляется по фактическому значению за указанный промежуток времени. Агрегированные значения могут быть средним, минимальным или максимальным значением. Для поля "Статистическое выражение" выберите значение Avg, Max или Min.

    ![Просмотр диаграммы](media/data-box-gateway-view-metrics/view-metrics-view-chart.png)

5. Если к выбранной метрике относятся несколько экземпляров, можно разделить их. Щелкните **Применить разделение**, а затем выберите значение, по которому нужно выполнить разбивку.

    ![Применение разделения](media/data-box-gateway-view-metrics/view-metrics-apply-splitting.png)

6. Теперь, если вам потребуется открыть распределение только для нескольких экземпляров, данные можно отфильтровать. Например, предположим, что вас интересует пропускная способность сети только для двух подключенных сетевых интерфейсов на вашем устройстве. Вы можете отфильтровать эти интерфейсы. Выберите **Добавить фильтр** и укажите имя сетевого интерфейса, который будете фильтровать.

    ![Добавление фильтра](media/data-box-gateway-view-metrics/view-metrics-add-filter.png)

7. Вы можете закрепить диаграмму на панели мониторинга для быстрого доступа к ней.

    ![Закрепить на панели мониторинга](media/data-box-gateway-view-metrics/view-metrics-pin-to-dashboard.png)

8. Чтобы экспортировать данные диаграммы в электронную таблицу Excel или получить ссылку, которая позволяет предоставить диаграмму другим пользователям, выберите элемент общего доступа на панели команд.

    ![Экспорт данных](media/data-box-gateway-view-metrics/view-metrics-export-data.png)

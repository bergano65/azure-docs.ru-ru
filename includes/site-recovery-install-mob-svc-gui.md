1. Скопируйте установку на сервер и откройте установщик.
2. В колонке **параметров установки** выберите **Install Mobility Service** (Установить Mobility Service).

    ![Страница параметров установки Mobility Service](./media/site-recovery-install-mob-svc-gui/mobility1.png)

3. Выберите расположение установки и щелкните **Установить**, чтобы начать процедуру установки.

    ![Страница параметров установки Mobility Service](./media/site-recovery-install-mob-svc-gui/mobility2.png)

4. Чтобы следить за ходом установки, используйте страницу **хода выполнения установки**.

    ![Страница хода выполнения установки Mobility Service](./media/site-recovery-install-mob-svc-gui/mobility3.png)

5. После завершения установки нажмите кнопку **Proceed to Configuration** (Перейти к конфигурации), чтобы зарегистрировать Mobility Service на сервере конфигурации.

    ![Страница регистрации Mobility Service](./media/site-recovery-install-mob-svc-gui/mobility4.png)

6. Выберите **Зарегистрировать**, чтобы завершить регистрацию.

    ![Страница завершения регистрации Mobility Service](./media/site-recovery-install-mob-svc-gui/mobility5.png)

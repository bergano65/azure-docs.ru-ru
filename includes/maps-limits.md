Ниже приведено ограничение использования для Azure Maps ценовой категории S0. Ограничение использования зависит от ценовой категории.

| Ресурс                              | Ограничение для ценовой категории S0 |
|---------------------------------------|:---------------------:|
| Максимальная частота запросов для подписки |   50 запросов в секунду  |

В следующей таблице показано ограничение совокупного размера данных для учетных записей Azure Maps в подписке Azure. Служба "Данные" Azure Maps доступна только в ценовой категории S1.

| Ресурс                               | Ограничение  |
|----------------------------------------|:---------------------:|
| Максимальный объем хранилища на подписку Azure |   1 ГБ |
| Максимальный размер файла для отправки           | 100 МБ |


См. сведения о [ценовых категориях Azure Maps](https://azure.microsoft.com/pricing/details/azure-maps/).

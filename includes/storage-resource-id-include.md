Идентификатор ресурса Azure AD указывает аудиторию, для которой выдаваемый маркер можно использовать для предоставления доступа к ресурсу Azure. В случае с хранилищем Azure идентификатор ресурса может быть указан отдельно для одной учетной записи хранения или может применяться к любой учетной записи хранения. В следующей таблице описаны значения, которые можно указать для идентификатора ресурса.

|Идентификатор ресурса  |Описание  |
|---------|---------|
|`https://<account>.blob.core.windows.net` <br /><br /> `https://<account>.queue.core.windows.net`    | Конечная точка службы для заданной учетной записи хранения. Используйте это значение, чтобы получить маркер для авторизации запросов к этой учетной записи хранения Azure и службе. Замените значение в квадратных скобках на имя вашей учетной записи хранения.      |
|`https://storage.azure.com/`     | Используйте, чтобы получить маркер для авторизации запросов к любой учетной записи хранения Azure.        |

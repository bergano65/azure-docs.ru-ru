## <a name="sign-in-to-azure"></a>Вход в Azure

Прежде чем публиковать приложение, необходимо войти в Azure.

1. Если вы не выполнили вход, щелкните значок Azure на панели действий, а затем в области **Azure: Функции** выберите **Войти в Azure**. Если у вас нет бесплатной учетной записи Azure, вы можете **[создать ее](https://azure.microsoft.com/free/)** . Учащиеся могут **[создать бесплатную учетную запись Azure для учащихся](https://azure.microsoft.com/free/students/)** .

    ![Вход в Azure через VS Code](./media/functions-sign-in-vs-code/functions-sign-into-azure.png)

    Если вы уже выполнили вход, перейдите к следующему разделу.

1. При появлении запроса в браузере выберите учетную запись Azure и войдите в нее с помощью учетных данных.

1. Выполнив вход, можно закрыть новое окно браузера. Подписки, принадлежащие учетной записи Azure, отображаются на боковой панели.

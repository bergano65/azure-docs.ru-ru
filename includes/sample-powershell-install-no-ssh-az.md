Для этого примера требуется Azure PowerShell. Чтобы узнать версию, выполните команду `Get-Module -ListAvailable Az`.
Если вам необходимо выполнить установку или обновление, см. статью [об установке модуля Azure PowerShell](/powershell/azure/install-az-ps). 

Запустите командлет [Connect AzAccount](/powershell/module/az.accounts/connect-azaccount), чтобы подключиться к Azure.

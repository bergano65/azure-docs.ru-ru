# <a name="create-ip-groups"></a>Создание групп IP-адресов

Группы IP позволяют группировать IP-адреса и управлять ими для правил брандмауэра Azure. Они могут иметь один IP-адрес, несколько IP-адресов или один или несколько диапазонов IP-адресов.

## <a name="create-an-ip-group---azure-portal"></a>Создание группы IP-адресов — портал Azure

Чтобы создать группу IP-адресов с помощью портал Azure, выполните следующие действия.

1. На домашней странице портал Azure выберите **создать ресурс**.
1. В поле поиска введите **IP-группы** , а затем выберите **IP-группы**.
1. Щелкните **Создать**.
1. Выберите свою подписку.
1. Выберите группу ресурсов или создайте новую.
1. Введите уникальное имя для IP-группы, а затем выберите регион.
1. По завершении выберите **Next: IP-адреса**.
1. Введите IP-адрес, несколько IP-адресов или диапазоны IP-адресов.

   Существует два способа ввода IP-адресов.
   - Их можно ввести вручную
   - Их можно импортировать из файла.

   Чтобы выполнить импорт из файла, выберите **Импорт из файла**. Вы можете либо перетащить файл в поле, либо выбрать пункт **Обзор файлов**. При необходимости можно просматривать и изменять отправленные IP-адреса.

   При вводе IP-адреса портал проверяет его на наличие перекрывающихся, повторяющихся и форматированных проблем.

1. По завершении выберите **проверить и создать**.
1. Щелкните **Создать**.

## <a name="create-an-ip-group---azure-powershell"></a>Создание группы IP-адресов — Azure PowerShell

В этом примере создается IP-группа с префиксом адреса и IP-адресом с помощью Azure PowerShell:

```azurepowershell
$ipGroup = @{
    Name              = 'ipGroup'
    ResourceGroupName = 'ipGroupRG'
    Location          = 'West US'
    IpAddress         = @('10.0.0.0/24', '192.168.1.10') 
}

New-AzIpGroup @ipGroup
```

## <a name="create-an-ip-group---azure-cli"></a>Создание группы IP-адресов — Azure CLI

В этом примере создается IP-группа с префиксом адреса и IP-адресом с помощью Azure CLI:

```azurecli-interactive
az network ip-group create \
    --name ipGroup \ 
    --resource-group ipGroupRG \
    --location westus \
    --ip-addresses '10.0.0.0/24' '192.168.1.10'
```

# <a name="azure-remote-rendering-support-options"></a>Варианты поддержки для удаленной подготовки к просмотру Azure

## <a name="open-a-tech-support-ticket"></a>Откройте запрос в службу технической поддержки

Чтобы открыть запрос в службу технической поддержки в портал Azure для удаленной подготовки к просмотру Azure:

1. Открыть [портал Azure](https://ms.portal.azure.com)

1. Выберите значок вопросительного знака в строке меню, а затем щелкните **Справка и поддержка**.

    ![портал Azure справки и поддержки](media/portal-help.png)

1. Выберите **Новый запрос в службу поддержки**.

    ![портал Azure новый запрос в службу поддержки](media/portal-new-request.png)

1. Заполните поля:

    * **Тип проблемы**:`Technical`
    * **Служба**:`Remote Rendering`

    ![Поля запроса в службу поддержки портал Azure](media/portal-request.png)

1. Нажмите кнопку **Далее: решения** в нижней части страницы.

1. Заполните приведенную ниже форму как точную и подробную, как можно Подробнее.

1. Проверьте и отправьте запрос в службу поддержки.

## <a name="community-support"></a>Поддержка сообщества

Сведения о поддержке из сообщества пользователей см. [в статье Удаленная визуализация Azure на Stack overflow](https://stackoverflow.com/questions/tagged/azure-remote-rendering).

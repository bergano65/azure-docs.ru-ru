# <a name="texconv---texture-conversion-tool"></a>Тексконв — инструмент преобразования текстур

Средство Тексконв — это внешнее средство командной строки для выполнения пакетных преобразований текстур.
Это средство можно найти в этом репозитории Git: [репозиторий GitHub для средства тексконв](https://github.com/microsoft/DirectXTex/wiki/Texconv).

Для удаленной отрисовки текстуры не нужно преобразовывать, поскольку шаг преобразования может работать с распространенными форматами файлов изображений и преобразовывать текстуры в соответствующий формат среды выполнения. Однако это средство может оказаться полезным для локального преобразования текстур с помощью настраиваемых параметров.

# <a name="whats-new-in-azure-load-balancer"></a>Новые возможности Azure Load Balancer

Azure Load Balancer регулярно обновляется. Будьте в курсе последних объявлений. В этой статье вы ознакомитесь:

- Последние выпуски.
- Известные проблемы
- Исправления ошибок
- со сведениями о нерекомендуемых функциях (если это применимо).

Кроме того, [здесь](https://azure.microsoft.com/updates/?category=networking&query=load%20balancer) вы найдете последние новости, касающиеся Azure Load Balancer, и сможете подписаться на канал RSS.

## <a name="recent-releases"></a>Последние выпуски

| Тип |name |Описание  |Дата добавления  |
| ------ |---------|---------|---------|
| Компонент | Поддержка перемещений между группами ресурсов | Load Balancer (цен. категория "Стандартный") и поддержка общедоступных IP-адресов (цен. категория "Стандартный") для [перемещения между группами ресурсов](https://azure.microsoft.com/updates/standard-resource-group-move/). | Октябрь 2020 г. |

Кроме того, можно резервировать IP-адреса в составе серверного пула до создания связанных ресурсов. Дополнительные сведения см. [здесь](backend-pool-management.md).|Июль 2020 г. |

Ранее такая конфигурация разрешалась, но не работала должным образом. Изменения в функциональные возможности внесены не были. Дополнительные сведения см. [здесь](load-balancer-ha-ports-overview.md#limitations).| Июнь 2020 г. |

Узнайте, как [создать приложение с двойным стеком](../virtual-network/virtual-network-ipv4-ipv6-dual-stack-standard-load-balancer-powershell.md). |Апрель 2020 г.|

## <a name="known-issues"></a>Известные проблемы

Группа разработчиков уже активно работает над решением следующих известных проблем:

|Проблема |Описание  |Меры по снижению риска  |
| ---------- |---------|---------|
| Экспорт Log Analytics | Log Analytics не может экспортировать метрики для Load Balancer (цен. категория "Стандартный") и журналы состояния пробы работоспособности для Load Balancer (цен. категория "Базовый")  | [Используйте Azure Monitor для сбора многомерных метрик для Load Balancer (цен. категория "Стандартный").](load-balancer-standard-diagnostics.md) Хотя вы не можете использовать Log Analytics для мониторинга, Azure Monitor обеспечивает визуализацию широкого спектра многомерных метрик. Вы можете использовать предварительно настроенную панель мониторинга метрик в подколонке "Аналитика" Load Balancer. Если вы используете Load Balancer (цен. категория "Базовый"), [обновитесь до цен. категории "Стандартный"](upgrade-basic-standard.md), чтобы обеспечить мониторинг метрик производственного уровня.

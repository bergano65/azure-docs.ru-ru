# <a name="remove-change-tracking-and-inventory-from-automation-account"></a>Удаление Отслеживание изменений и инвентаризации из учетной записи службы автоматизации

После включения управления виртуальными машинами с помощью службы автоматизации Azure Отслеживание изменений и инвентаризации вы можете отменить ее использование и удалить конфигурацию из учетной записи и связанной Log Analytics рабочей области. В этой статье объясняется, как полностью удалить Отслеживание изменений и инвентаризацию из управляемых виртуальных машин, учетной записи службы автоматизации и Log Analytics рабочей области.

## <a name="sign-into-the-azure-portal"></a>Войдите на портал Azure.

Войдите на [портал Azure](https://portal.azure.com).

## <a name="remove-management-of-vms"></a>Удаление управления виртуальными машинами

Перед удалением Отслеживание изменений и инвентаризации необходимо сначала прерывать Управление виртуальными машинами. Дополнительные сведения см. в статье [Удаление виртуальных машин из отслеживание изменений для отмены](remove-vms-from-change-tracking.md) их регистрации в компоненте.

## <a name="remove-changetracking-solution"></a>Удалить решение отслеживания изменений

Прежде чем удалить связь учетной записи службы автоматизации с рабочей областью, необходимо выполнить следующие действия для полного удаления Отслеживание изменений и инвентаризации. Вы удалите решение **отслеживания изменений** из рабочей области.

1. На портале Azure щелкните **Все службы**. В списке ресурсов введите **Log Analytics**. По мере ввода, список фильтрует предложения на основе введенных данных. Выберите **Log Analytics**.

2. В списке рабочих областей Log Analytics выберите рабочую область, которая была выбрана при включении Отслеживание изменений и инвентаризации.

3. В левой части выберите **решения**.  

4. В списке решений выберите **отслеживания изменений (имя рабочей области)**. На странице **Обзор** для решения выберите **Удалить**. При появлении запроса на подтверждение выберите **Да**.

## <a name="unlink-workspace-from-automation-account"></a>Отмена связи рабочей области с учетной записью службы автоматизации

1. На портале Azure выберите раздел **Учетные записи службы автоматизации**.

2. Откройте страницу учетной записи службы автоматизации и выберите **Связанная рабочая область** в разделе **Связанные ресурсы**.

3. На странице **Удалить связь с рабочей областью** выберите **отсоединить рабочую область** и ответьте на запросы.

   ![Страница "Отмена связи с рабочей областью"](media/remove-feature/automation-unlink-workspace-blade.png)

При попытке удалить связь с рабочей областью Log Analytics можно отслеживать ход выполнения в разделе **уведомления** в меню.

Кроме того, связь рабочей области Log Analytics с учетной записью службы автоматизации можно удалить прямо из рабочей области:

1. На портале Azure выберите **Log Analytics**.

2. В рабочей области выберите **учетная запись службы автоматизации** в разделе **связанные ресурсы**.

3. На странице учетной записи службы автоматизации выберите **Отменить связь с учетной записью**.

При попытке отменить связь с учетной записью службы автоматизации ход выполнения можно отслеживать в разделе **уведомления** в меню.

# <a name="machine-learning-on-hdinsight"></a>Машинное обучение в HDInsight

HDInsight позволяет выполнять машинное обучение с помощью больших данных, предоставляя возможность получить ценные сведения из больших объемов (петабайтов, а может дальше эксабайтов) структурированных, неструктурированных и быстро меняющихся данных. Существует несколько вариантов машинного обучения в HDInsight: SparkML и Apache Spark MLlib, R, Apache Hive и Microsoft Cognitive Toolkit.

## <a name="sparkml-and-mllib"></a>SparkM и MLlib

[HDInsight Spark](spark/apache-spark-overview.md) — это размещаемая в Azure унифицированная платформа параллельной обработки данных [Apache Spark](https://spark.apache.org/) с открытым кодом, которая поддерживает обработку в памяти для оптимизации анализа больших данных. Подсистема обработки Spark призвана ускорить разработку, повысить удобство использования и реализовать сложную аналитику. Возможности распределенного вычисления в памяти Spark отлично подходят для итеративных алгоритмов, используемых в машинном обучении и графовых вычислениях. Существуют двt масштабируемых библиотеки машинного обучения Spark, которые предоставляют возможности алгоритмического моделирования для этой распределенной среды: SparkM и MLlib. MLlib содержит исходный API, созданный на основе RDD. SparkML — это новый пакет, который предоставляет API более высокого уровня, созданный поверх кадров данных для построения конвейеров машинного обучения. SparkML еще не поддерживает все функции MLlib, но заменяет MLlib как стандартную библиотеку машинного обучения Spark.

Библиотекой службы "Машинное обучение Microsoft" для Apache Spark является [MMLSpark](https://github.com/Azure/mmlspark). Эта библиотека предназначена для оптимизации работы специалистов по обработке и анализу данных в Spark, ускоренной реализации экспериментов и использования самых современных методов машинного обучения, включая глубокое обучение, на очень больших наборах данных. MMLSpark предоставляет слой поверх низкоуровневых API SparkML при построении масштабируемых моделей ML, таких как индексирование строк, принудительное преобразование данных в макет, ожидаемый алгоритмами машинного обучения, и сборки векторов признаков. Библиотека MMLSpark упрощает эти и другие общие задачи моделирования для создания моделей в PySpark.

## <a name="r"></a>R

В настоящее время [R](https://www.r-project.org/) является наиболее популярным статистическим языком программирования в мире. Это средство визуализации данных с открытым исходным кодом, сообществом более 2 500 000 пользователей и увеличивающим их. Благодаря своей богатой пользовательской базе и более чем 8000 опубликованных пакетов R является хорошим выбором для многих компаний, которым требуется машинное обучение. Вы можете создать кластер HDInsight со службами машинного обучения, который можно использовать с большими наборами данных и моделями. Эта возможность предоставляет специалистам по обработке и анализу данных, а также статистикам знакомый интерфейс R, который может масштабироваться по требованию через HDInsight без дополнительных затрат на настройку и обслуживание кластера.

![Обучение прогнозированию с помощью R Server](./media/hdinsight-machine-learning-overview/training-for-prediction.png)

Для подключения к кластеру и выполнения скриптов на языке R удобно использовать граничный узел кластеров.  Можно также выполнять скрипты R на узлах кластера, используя контексты вычислений Hadoop Scale для уменьшения или Spark.

Используя службы машинного обучения в HDInsight со Spark, вы можете параллелизировать обучение на узлах кластера с помощью контекста вычислений Spark. Сценарии R можно выполнять на граничном узле, используя все доступные ядра в параллельном режиме, если это необходимо. Кроме того, можно выполнить код с граничного узла, чтобы запустить обработку, которая распределяется по всем узлам в кластере. Службы ML в HDInsight с Spark также позволяют реализовать функции параллелизации из пакетов R с открытым кодом, если это необходимо.

## <a name="azure-machine-learning-and-apache-hive"></a>Машинное обучение Azure и Apache Hive

Машинное обучение Azure предоставляет средства для моделирования прогнозной аналитики и полностью управляемую службу, которую можно использовать для развертывания прогнозных моделей как готовых к использованию веб-служб. Служба машинного обучения Azure является полноценным решением прогнозирующей аналитики в облаке, которое используется для создания, тестирования, развертывания и контроля моделей прогнозирования. Выбирайте из большой библиотеки алгоритмов, используйте веб-студию для создания моделей и с легкостью развертывайте свою модель в виде веб-службы.

![Обзор Microsoft Azure машинного обучения](./media/hdinsight-machine-learning-overview/azure-machine-learning.png)

Создайте характеристики для данных в кластере Hadoop с помощью [запросов Hive](../machine-learning/team-data-science-process/create-features-hive.md). *Конструирование характеристик* пытается повысить эффективность прогнозирования алгоритмов обучения путем создания характеристик из необработанных данных, позволяющих упростить процесс обучения. Вы можете выполнять запросы HiveQL из Машинное обучение Azure Studio (классическая модель) и обращаться к данным, обработанным в Hive, и хранить их в хранилище BLOB-объектов с помощью [модуля импорт данных](../machine-learning/studio/import-data.md).

## <a name="microsoft-cognitive-toolkit"></a>Microsoft Cognitive Toolkit

[Глубокое обучение](https://www.microsoft.com/en-us/research/group/dltc/) является типом машинного обучения, использующим нейронные сети, построенные по принципу биологических процессов в человеческом мозгу. Многие исследователи рассматривают глубокое обучение как перспективный подход к улучшению искусственного интеллекта. Примерами глубокого обучения являются программы-переводчики разговорного языка, системы распознавания образов и механизмы машинного рассуждения.

Чтобы оптимизировать свою работу в области глубокого обучения, корпорация Майкрософт разработала бесплатный простой в использовании набор средств с открытым исходным кодом — [Microsoft Cognitive Toolkit](https://www.microsoft.com/en-us/cognitive-toolkit/). Он широко используется многими продуктами Microsoft, компаниями во всем мире, которым необходимо развертывать глубокое обучение в масштабе, а также учащимися, заинтересованными в современных алгоритмах и методах.

# <a name="validate-a-dynamic-group-membership-rule-preview-in-azure-active-directory"></a>Проверка правила членства в динамической группе (Предварительная версия) в Azure Active Directory

Azure Active Directory (Azure AD) теперь предоставляет средства для проверки динамических правил групп (в общедоступной предварительной версии). На вкладке **Проверка правил** можно проверить динамическое правило по образцу членов группы, чтобы убедиться, что правило работает должным образом. При создании или обновлении динамических групп правил администраторам необходимо определить, будет ли пользователь или устройство членом группы. Это помогает определить, соответствует ли пользователь или устройство условиям правила, и помочь в устранении неполадок, когда членство не ожидается.

## <a name="step-by-step-walk-through"></a>Пошаговое руководство

Чтобы приступить к работе, перейдите в раздел **Azure Active Directory**  >  **группы**. Выберите существующую динамическую группу или создайте новую динамическую группу и щелкните правила динамического членства. После этого можно просмотреть вкладку **Проверка правил** .

![Найдите вкладку Проверка правил и начните с существующего правила.](./media/groups-dynamic-rule-validation/validate-tab.png)

На вкладке **Проверка правил** можно выбрать пользователей для проверки членства. 20 пользователей или устройств можно выбрать одновременно.

![Добавление пользователей для проверки существующего правила](./media/groups-dynamic-rule-validation/validate-tab-add-users.png)

После выбора пользователей или устройств в средстве выбора и **выбора**, проверка будет автоматически запущена, и отобразятся результаты проверки.

![Просмотр результатов проверки правила](./media/groups-dynamic-rule-validation/validate-tab-results.png)

Результаты показывают, является ли пользователь членом группы. Если правило недопустимо или возникла сетевая ошибка, результат будет отображаться как **неизвестный**. В случае **неизвестной** ошибки в подробном сообщении об ошибке описываются проблемы и необходимые действия.

![Просмотр сведений о результатах проверки правила](./media/groups-dynamic-rule-validation/validate-tab-view-details.png)

Вы можете изменить правило и проверку членства. Чтобы узнать, почему пользователь не является членом группы, щелкните "просмотреть сведения" и сведения о проверке появятся результат каждого выражения, формирующего правило. Нажмите кнопку **ОК** , чтобы выйти.

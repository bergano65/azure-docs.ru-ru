# <a name="view-and-assign-administrator-roles-in-azure-active-directory"></a>Просмотр и назначение ролей администратора в Azure Active Directory

Теперь вы можете просматривать всех членов роли администратора на портале Azure Active Directory и управлять ими. Если вы часто управляете назначениями ролей, возможно, вам пригодится эта функция. Если вы когда-нибудь задумывались, для чего нужны все эти роли, просмотрите подробный список разрешений для каждой роли администратора в Azure AD.

## <a name="view-all-roles"></a>Просмотр всех ролей

1. Войдите в [портал Azure](https://portal.azure.com) и выберите **Azure Active Directory**.

1. Выберите **роли и администраторы** , чтобы просмотреть список всех доступных ролей.

1. Нажмите кнопку с многоточием справа от каждой строки, чтобы просмотреть разрешения для роли. Выберите роль, чтобы просмотреть пользователей, назначенных роли. Если вы видите что-то другое на следующем рисунке, прочитайте примечание в разделе [Просмотр назначений для привилегированных ролей](#view-assignments-for-privileged-roles) , чтобы проверить, находитесь ли вы в Управление привилегированными пользователями (PIM).

    ![Список ролей на портале Azure AD](./media/directory-manage-roles-portal/view-roles-in-azure-active-directory.png)

## <a name="view-my-roles"></a>Просмотр своих ролей

Вы также можете просмотреть и свои разрешения. Выберите **Ваша роль** на странице **Роли и администраторы**, чтобы увидеть роли, назначенные вам сейчас.

## <a name="view-assignments-for-privileged-roles"></a>Просмотр назначений для привилегированных ролей

Нажмите **Управление в PIM**, чтобы открыть дополнительные возможности управления. Администраторы привилегированных ролей могут менять "постоянные" назначения (пользователь всегда выступает в этой роли) на "разрешенные" (роль назначается в отдельных случаях). Если у вас нет управление привилегированными пользователями, вы по-прежнему можете выбрать **Управление в PIM** , чтобы зарегистрироваться для получения пробной версии. Для управления привилегированными пользователями (PIM) требуется [план лицензирования Azure AD Premium P2](../privileged-identity-management/subscription-requirements.md).

![Список членов роли администратора](./media/directory-manage-roles-portal/member-list.png)

Если вы являетесь глобальным администратором или администратором привилегированных ролей, вы можете добавлять или удалять участников, отфильтровывать список или выбирать участников, чтобы просмотреть их активные назначенные роли.

> [!Note]
> Если у вас есть лицензия Azure AD Premium P2 и вы уже используете управление привилегированными пользователями, все задачи управления ролями выполняются в управлении удостоверениями прав, а не в Azure AD.
>
> ![Роли Azure AD, управляемые в PIM для пользователей, которые уже используют PIM и имеют лицензию Premium P2](./media/directory-manage-roles-portal/pim-manages-roles-for-p2.png)

## <a name="view-a-users-role-permissions"></a>Просмотр разрешений роли для пользователя

При просмотре членов роли нажмите **Описание**, чтобы просмотреть полный список разрешений для этой роли. Страница содержит ссылки на соответствующую документацию со сведениями об управлении ролями в каталоге.

![список разрешений для роли администратора](./media/directory-manage-roles-portal/role-description.png)

## <a name="assign-a-role"></a>Назначение роли

1. Войдите в [портал Azure](https://portal.azure.com) с разрешениями администратора глобального администратора или привилегированной роли и выберите **Azure Active Directory**.

1. Выберите **роли и администраторы** , чтобы просмотреть список всех доступных ролей.

1. Выберите роль, чтобы просмотреть ее назначения.

    ![список разрешений для роли администратора](./media/directory-manage-roles-portal/member-list.png)

1. Выберите **добавить назначения** и выберите роли, которые необходимо назначить. Нажмите **Управление в PIM**, чтобы открыть дополнительные возможности управления. Если вы видите что-то другое на следующем рисунке, прочитайте примечание в разделе [Просмотр назначений для привилегированных ролей](#view-assignments-for-privileged-roles) , чтобы проверить, находитесь ли вы в PIM.

    ![список разрешений для роли администратора](./media/directory-manage-roles-portal/directory-role-select-role.png)

# <a name="view-and-assign-administrator-roles-in-azure-active-directory"></a>Просмотр и назначение ролей администратора в Azure Active Directory

Теперь вы можете просматривать всех членов роли администратора на портале Azure Active Directory и управлять ими. Если вы часто управляете назначениями ролей, возможно, вам пригодится эта функция. Если вы когда-нибудь задумывались, для чего нужны все эти роли, просмотрите подробный список разрешений для каждой роли администратора в Azure AD.

## <a name="view-all-roles"></a>Просмотр всех ролей

В Azure Active Directory выберите **Роли и администраторы**, чтобы просмотреть список всех доступных ролей. 

Нажмите на многоточие справа от каждой строки, чтобы открыть подробное описание роли.

![Список ролей на портале Azure AD](./media/directory-manage-roles-portal/role-list.png)

## <a name="view-my-roles"></a>Просмотр своих ролей

Вы также можете просмотреть и свои разрешения. Выберите **Ваша роль** на странице **Роли и администраторы**, чтобы увидеть роли, назначенные вам сейчас.

## <a name="view-assignments-for-a-role"></a>Просмотр сведений о назначениях для роли

Выберите роль, чтобы просмотреть список пользователей, которым назначена эта роль. Нажмите **Управление в PIM**, чтобы открыть дополнительные возможности управления. Администраторы привилегированных ролей могут менять "постоянные" назначения (пользователь всегда выступает в этой роли) на "разрешенные" (роль назначается в отдельных случаях). Если у вас нет PIM, вы можете нажать **Управление в PIM**, чтобы зарегистрироваться для получения пробной версии. Для управления привилегированными пользователями (PIM) требуется [план лицензирования Azure AD Premium P2](../privileged-identity-management/subscription-requirements.md).

![Список членов роли администратора](./media/directory-manage-roles-portal/member-list.png)

Если вы являетесь глобальным администратором или администратором привилегированных ролей, вы можете добавлять или удалять участников, отфильтровывать список или выбирать участников, чтобы просмотреть их активные назначенные роли.

## <a name="view-a-users-role-permissions"></a>Просмотр разрешений роли для пользователя

При просмотре членов роли нажмите **Описание**, чтобы просмотреть полный список разрешений для этой роли. Страница содержит ссылки на соответствующую документацию со сведениями об управлении ролями в каталоге.

![список разрешений для роли администратора](./media/directory-manage-roles-portal/role-description.png)

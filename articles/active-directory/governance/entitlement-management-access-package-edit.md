# <a name="hide-or-delete-an-access-package-in-azure-ad-entitlement-management"></a>Скрытие или удаление пакета Access в управлении назначениями Azure AD

Пакеты доступа по умолчанию обнаруживаются. Это означает, что если политика разрешает пользователю запрашивать пакет доступа, он автоматически увидит пакет Access, указанный на портале моего доступа. Однако можно изменить скрытый параметр таким образом, чтобы пакет Access не **перекрывались** на портале "мой доступ" пользователя.

В этой статье описывается, как скрыть или удалить пакет Access.

## <a name="change-the-hidden-setting"></a>Изменение скрытого параметра

Выполните следующие действия, чтобы изменить **скрытый** параметр для пакета доступа.

**Требуемая роль:** Глобальный администратор, Администратор пользователей, Владелец каталога или Диспетчер пакетов для доступа.

1. На портале Azure щелкните **Azure Active Directory** и выберите **Управление удостоверениями**.

1. В меню слева щелкните **доступ к пакетам** и откройте пакет Access.

1. На странице Обзор нажмите кнопку **изменить**.

1. Установите параметр **Hidden** .

    Если задано значение **нет**, пакет Access будет указан на портале "мой доступ" пользователя.

    Если задано значение **"Да"**, пакет Access не будет отображаться на портале пользователя "мой доступ". Единственный способ просмотреть пакет Access доступен пользователю, если у него есть ссылка на портал прямой **доступ** к пакету Access. Дополнительные сведения см. [в разделе Совместное использование ссылки для запроса пакета Access](entitlement-management-access-package-settings.md).

## <a name="delete-an-access-package"></a>Удаление пакета Access

Пакет Access можно удалить, только если в нем нет активных назначений пользователей. Выполните следующие действия, чтобы удалить пакет Access.

**Требуемая роль:** Глобальный администратор, Администратор пользователей, Владелец каталога или Диспетчер пакетов для доступа.

1. На портале Azure щелкните **Azure Active Directory** и выберите **Управление удостоверениями**.

1. В меню слева щелкните **доступ к пакетам** и откройте пакет Access.

1. В меню слева щелкните **назначения** и удалите доступ для всех пользователей.

1. В меню слева щелкните **Обзор** , а затем нажмите кнопку **Удалить**.

1. В появившемся сообщении об удалении нажмите кнопку **Да**.

# <a name="validation-differences-by-supported-account-types-signinaudience"></a>Различия в проверке по поддерживаемым типам счетов (Сигнинаудиенце)

При регистрации приложения с помощью платформы идентификации Майкрософт для разработчиков вам будет предложено выбрать типы учетных записей, которые поддерживает ваше приложение. В объекте приложения и манифесте это свойство имеет значение `signInAudience`.

Возможны следующие варианты:

- *Азуреадмйорг*: только учетные записи в каталоге Организации, в котором зарегистрировано приложение (один клиент)
- *Азуреадмултиплеоргс*: учетные записи в любом организационном каталоге (несколько клиентов)
- *Азуреадандперсоналмикрософтаккаунт*: учетные записи в любом организационном каталоге (с несколькими клиентами) и в личных учетных записях Майкрософт (например, Skype, Xbox и Outlook.com).

Для зарегистрированных приложений можно найти значения для поддерживаемых типов учетных записей в разделе **Проверка подлинности** приложения. Его также можно найти в свойстве `signInAudience` в **манифесте**.

Значение, выбранное для этого свойства, влияет на другие свойства объекта приложения. В результате, если изменить это свойство, может потребоваться сначала изменить другие свойства.

# <a name="manage-organizations-you-have-access-to-in-the-my-account-portal"></a>Управление организациями, к которым у вас есть доступ на портале моей учетной записи

Страница **Организации** на портале " **Моя учетная запись** " помогает управлять организациями (как правило, названием вашей организации), к которым у вас есть доступ. На странице **Организации** отображаются два типа организаций.

- **Домашняя организация.** Ваша домашняя организация — это организация, которой принадлежит ваша рабочая или учебная учетная запись. Вы не можете покинуть домашнюю организацию.

    >[!NOTE]
    > Если у вас нет назначенной домашней организации, отобразится только один заголовок "Организации" со списком связанных организаций.

- **Другие организации.** Другие организации — это любая группа, в которую вы вошли, прежде чем использовать вашу рабочую или учебную учетную запись. Вы можете покинуть эти организации в любое время.

>[!Important]
>Эта статья предназначена для пользователей, пытающихся обновить сведения об организации, к которым обращается Рабочая или учебная учетная запись. Если вы являетесь администратором, которому требуются сведения об управлении группами и пользователями для своих сотрудников и других целей, обратитесь к статье [Руководство по управлению корпоративными пользователями](../enterprise-users/index.yml).

## <a name="view-your-organizations"></a>Просмотр ваших организаций

1. Войдите в рабочую или учебную учетную запись, а затем перейдите в раздел **Моя учетная запись** (страница https://myaccount.microsoft.com/) ).

2. Выберите на панели навигации слева **Организации** или из блока **Организации** выберите ссылку **Управление организациями**.

    ![Выделенные ссылки на организации на странице "Моя учетная запись"](media/my-account-portal/my-account-portal-organizations.png)

3. Проверьте информацию о вашей **домашней организации**.

    ![Страница "Организации"](media/my-account-portal/my-account-portal-organization-page.png)

4. Ознакомьтесь с другими организациями, убедившись, что вы знаете все организации, к которым у вас есть доступ.

## <a name="leave-an-organization"></a>Выход из организации

Вы можете выйти из любой организации, кроме домашней.

>[!Important]
>Если вы случайно вышли из организации, нельзя это отменить. Вам придется подождать, пока администратор этой организации не отправит вам приглашение.

- Выберите **Покинуть организацию** напротив организации, которую вы хотите покинуть, а затем — **Да**, чтобы подтвердить выход из организации.

    ![Страница "Организации" со ссылкой на выход из организации](media/my-account-portal/my-account-portal-organizations-leave.png)

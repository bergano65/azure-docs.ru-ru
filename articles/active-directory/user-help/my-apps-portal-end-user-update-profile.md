# <a name="update-your-profile-and-account-info-from-the-my-apps-portal"></a>Обновить сведения профиля и учетной записи на портале Мои приложения
Вашей рабочей или учебной учетной записи можно использовать с веб интерфейс **Мои приложения** портала, просматривать и запускать многие из вашей организации облачных приложений, чтобы обновить некоторые данные профиля и учетной записи, чтобы увидеть ваши **группы** сведения, а также для выполнения **проверок доступа** для приложений и групп. Если у вас нет доступа к **Мои приложения** портала, необходимо обратиться в службу поддержки для разрешения.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-my-apps-portal.md)]

>[!Important]
>Эти материалы предназначены для пользователей. Если вы являетесь администратором, можно найти дополнительные сведения о том, как создавать и настраивать облачные приложения в [приложениями](https://docs.microsoft.com/azure/active-directory/manage-apps).

## <a name="view-your-organization-related-profile-information"></a>Просматривать данные вашего профиля, относящиеся к организации
В зависимости от вашей организации она выбрана для включения в **профиль** странице **Мои приложения** портала, может появиться в конкретные подробности относящимися к работе, вместе с устройствами, действий и любые Дополнительные организации, которой принадлежит пользователь.

### <a name="to-view-your-profile-information"></a>Чтобы просмотреть данные профиля
1.  Войдите свою учетную запись рабочей или учебной, [перейдите **Мои приложения** портала](my-apps-portal-end-user-access.md).

2. Из **приложений** выберите изображение профиля в правом верхнем углу, где отображается надпись свое имя и вашей организации, и затем выберите **профиль**.

    **Профиль** появится страница, показывающая данные профиля.

    ![Отображается страница профиля, все доступные сведения](media/my-apps-portal/my-apps-portal-profile-page.png)

3. Из **профиль** страницы, вы можете:

    - **Просмотрите свои данные, относящиеся к организации.** Просмотр фотографий, имя, заголовок, адреса электронной почты связан и сведений об аудитории. Эта информация управляется вашей организацией, и вы не сможете изменить его. Если появится сообщение об ошибке, обратитесь в службу поддержки.

    - **Просмотрите устройства и действия**. Убедитесь, что каждое устройство, знакомы и правильности их подключения к вашей организации. Если вы не узнаете устройства, выберите **отключить устройство** удалить связь с учетной записью. После отключения устройства, он удаляется из этой страницы.

    - **Просмотрите вашей организации.** Убедитесь, что вы по-прежнему работать с каждым из указанного организаций. Если вы больше не работает с организацией, настоятельно рекомендуется выбрать **войдите покинуть организацию**. После выхода из организации, он удаляется из этой страницы.

## <a name="manage-your-work-or-school-account-information"></a>Рабочей или учебной учетных записей
Обновление и рабочей или учебной учетной записи из данных **профиль** странице **Мои приложения** портала. Эта страница поможет вам:

- Измените рабочую или учебную пароль учетной записи.

- Включите пароль сброса (если администратор отключил эту функцию).

- Укажите сведения о проверке для обеспечения дополнительной безопасности.

- Просмотрите условия использования вашей организации.

- Выйдите из везде.

## <a name="change-your-password"></a>Смена пароля
Если вы хотите изменить пароль для вашей рабочей или учебной учетной записи, можно выбрать **Смена пароля** из **Управление учетной записью** область **профиль** страницы.

### <a name="to-change-your-password"></a>Чтобы изменить пароль
1. Из **Управление учетной записью** область **профиль** выберите **Смена пароля**.

2. На **Смена пароля** странице, проверьте правильность идентификатора пользователя и введите старый пароль и новый пароль в поля.

    ![Страница смены пароля](media/my-apps-portal/my-apps-portal-change-password-page.png)

3. Нажмите кнопку **Submit** (Отправить).

    Ваш пароль изменен. Вам будет необходимо войти в любых приложениях, которые вы ранее вошли с помощью рабочей или учебной учетной записи.

## <a name="set-up-and-use-password-reset"></a>Установить и использовать сброс пароля
Если вы забыли пароль, никогда не от вашей компании или были заблокированы из учетной записи, можно сбросить свой пароль.

>[!Important]
>Администратору необходимо включить эту функцию, и необходимо зарегистрироваться для участия. Дополнительные сведения о регистрации и сброса пароля см. в разделе [регистрация для самостоятельного сброса пароля](active-directory-passwords-reset-register.md) и [сбросить пароль рабочей или учебной](user-help-reset-password.md).

## <a name="change-your-security-verification-information"></a>Изменить параметры проверки безопасности
Если ваша организация требует использовать двухфакторную проверку, можно Добавление, обновление и удаление сведений о безопасности, связанные с **дополнительная проверка безопасности** страницы.

Двухфакторной проверки требуется два компонента проверки сведения, такие как пароль и ПИН-код, перед получением информации организации или учетной записи. Дополнительные сведения о двухфакторной проверки, см. в разделе [Настройка учетной записи для двухфакторной проверки подлинности](multi-factor-authentication-end-user-first-time.md).

### <a name="to-change-your-security-information"></a>Для изменения сведений о безопасности
1. Из **Управление учетной записью** область **профиль** выберите **дополнительная проверка безопасности**.

    ![Странице дополнительной проверки безопасности](media/my-apps-portal/my-apps-portal-additional-verification-page.png)

2. На **дополнительная проверка безопасности** странице, можно добавить, изменить или удалить следующие сведения:

    - **Параметр проверки по умолчанию.** Выбор дополнительного метода по умолчанию для двухфакторной проверки. Этот метод автоматически используется каждый раз, когда необходим двухфакторную проверку, после того как вы введете имя пользователя и пароль.

    - **Добавление, обновление или удаление методов проверки.** Можно добавить новую, обновить существующие или удалять старые сведения о, которое больше не применяется.

    - **Настройте приложение Microsoft Authenticator.** Вы можете настроить приложение Microsoft Authenticator для работы в качестве способа проверки. Дополнительные сведения о приложении Microsoft Authenticator, см. в разделе [что такое приложение Microsoft Authenticator?](user-help-auth-app-overview.md)

3. Выберите **Сохранить**, чтобы сохранить изменения.

## <a name="review-your-organizations-terms-of-use-statement"></a>Ознакомьтесь с условиями вашей организации инструкция use
Вы можете ознакомиться с условиями вашей организации, используйте инструкции, если таковой имеется.

1. Из **Управление учетной записью** область **профиль** выберите **ознакомьтесь с условиями использования**.

2. Ознакомьтесь с условиями использования и выберите организации **Accept** чтобы убедиться, что вы прочитано и понято условия использования для вашей организации.

    ![Условия использования страница портала Мои приложения](media/my-apps-portal/my-apps-portal-tou-page.png)

    Если у вашей организации нет условия использования, можно выбрать **сделать** для возврата **профиль** страницы.

## <a name="sign-out-of-everywhere"></a>Выйдите из везде
Вы можете выйти из везде, где вы вошли с помощью рабочей или учебной учетной записи. Сюда входят все приложения и устройства.

### <a name="to-sign-out-of-everywhere"></a>Чтобы выйти из везде
1. Из **Управление учетной записью** область **профиль** выберите **выйдите из системы everywhere**.

2. В **выйдите из системы everywhere** окно подтверждения, выберите **Да** для подтверждения того, что вы хотите выйти из всех сеансов и устройств. Выберите **нет** Если вы измените свое решение.

# <a name="sign-in-using-two-step-verification-or-security-info"></a>Вход с помощью двухфакторной проверки подлинности или сведений для защиты

После настройки двухфакторной проверки подлинности или сведений о безопасности вы сможете войти в свою учетную запись с помощью указанного метода проверки.

> [!Note]
> Если вы по-прежнему используете двухфакторную проверку подлинности, необходимо настроить методы проверки, следуя инструкциям в статье [Настройка учетной записи для двухфакторной проверки](multi-factor-authentication-end-user-first-time.md) подлинности.
>
> Если администратор включил сведения о безопасности, необходимо настроить методы проверки с помощью этих пошаговых руководств:<ul><li>[Настройка использования приложения проверки подлинности в сведениях для защиты (предварительная версия)](security-info-setup-auth-app.md)</li><li>[Настройка сведений для защиты для использования обмена сообщениями](security-info-setup-text-msg.md)</li><li>[Настройка использования телефонного звонка в сведениях для защиты (предварительная версия)](security-info-setup-phone-number.md)</li><li>[Настройка сведений о безопасности для использования ключа безопасности](security-info-setup-security-key.md)</li></ul>

## <a name="sign-in-using-an-authenticator-app-notification-on-your-mobile-device"></a>Вход с помощью уведомления от приложения проверки подлинности на мобильном устройстве

1. Войдите в учетную запись, используя имя пользователя и пароль.

2. В уведомлении, которое отправилось на мобильное устройство, выберите **Утвердить**.

## <a name="sign-in-using-an-authenticator-app-code-on-your-mobile-device"></a>Вход с помощью кода в приложении проверки подлинности на мобильном устройстве

1. Войдите в учетную запись, используя имя пользователя и пароль.

2. Откройте приложение проверки подлинности и введите случайным образом сформированный код для своей учетной записи в поле **Введите код**.

## <a name="sign-in-using-your-phone-number"></a>Вход с использованием номера телефона

1. Войдите в учетную запись, используя имя пользователя и пароль.

2. Ответьте на вызов, а затем следуйте инструкциям.

## <a name="sign-in-using-a-text-message"></a>Вход с использованием текстового сообщения

1. Войдите в учетную запись, используя имя пользователя и пароль.

2. Откройте сообщение и введите код из текста в поле **Введите код**.

## <a name="sign-in-using-a-security-key-at-the-lock-screen"></a>Вход с помощью ключа безопасности на экране блокировки

1. После регистрации ключа безопасности выберите образ ключа безопасности на экране блокировки Windows 10.

2. Вставьте ключ безопасности в USB-порт устройства и войдите в Windows с помощью ПИН-кода ключа безопасности.

    ![Вход с помощью ключа безопасности на экране блокировки Windows 10](./media/security-info/security-info-windows-10-lock-screen-security-key.png)

## <a name="sign-in-using-a-security-key-and-the-microsoft-edge-browser"></a>Вход с помощью ключа безопасности и браузера Microsoft ребра

1. После регистрации ключа безопасности откройте браузер Microsoft ребр.

2. При появлении запроса на вход вставьте ключ безопасности в USB-порт устройства и войдите в Windows, используя ПИН-код ключа безопасности.

    ![Вход с использованием ключа безопасности с помощью браузера Microsoft ребр](./media/security-info/security-info-edge-security-key.png)

    >[!NOTE]
    >Сведения о входе в систему с помощью Microsoft Authenticatorного приложения см. в статье [Вход в учетные записи с помощью приложения Microsoft Authenticator](user-help-auth-app-sign-in.md).

## <a name="sign-in-using-another-verification-method"></a>Выполнение входа с использованием другого способа проверки

Если по какой-либо причине вы не можете использовать основной метод входа, можно использовать другой ранее настроенный метод проверки.

1. Войдите в учетную запись в обычном режиме, а затем щелкните ссылку Войти по ссылке **другой путь** на странице с **двумя этапными проверками** .

    ![Изменение метода проверки при входе](media/security-info/two-factor-auth-signin-another-way.png)

    >[!Note]
    >Если вы не видите ссылку **другой способ** , это означает, что вы не настроили другие методы проверки и вам придется обратиться к администратору для получения помощи при входе в свою учетную запись. После того как администратор попытается войти в систему, убедитесь, что добавлены дополнительные методы проверки. Дополнительные сведения о добавлении методов проверки см. в статье [Управление параметрами двухфакторной проверки подлинности](multi-factor-authentication-end-user-manage-settings.md).
    >
    >Если вы видите ссылку **другой способ** , но не видите никаких других методов проверки, обратитесь к администратору за помощью в входе в свою учетную запись.

2. Выберите альтернативный метод проверки подлинности и продолжайте процесс двухфакторной проверки подлинности.

3. При необходимости можно обновить свои методы проверки, как только вы вернетесь в свою учетную запись. Дополнительные сведения о добавлении или изменении методов см. в статье [Управление параметрами двухфакторной проверки подлинности](multi-factor-authentication-end-user-manage-settings.md).

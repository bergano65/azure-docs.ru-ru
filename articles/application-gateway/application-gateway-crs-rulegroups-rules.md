# <a name="web-application-firewall-crs-rule-groups-and-rules"></a>Группы правил и правила CR брандмауэра веб-приложения

Брандмауэр веб-приложения шлюза приложений (WAF) позволяет устранить уязвимости веб-приложений и защитить их от эксплойтов. Это делается с помощью правил, определенных на основе набора правил OWASP Core, наборов 3,0 или 2.2.9. Эти правила можно отключать по отдельности. В этой статье приведены текущие правила и наборы правил.

При использовании шлюза приложений с брандмауэром веб-приложения доступны следующие группы правил и правила.

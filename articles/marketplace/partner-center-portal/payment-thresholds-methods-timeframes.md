# <a name="payment-thresholds-methods-and-time-frames"></a>Пороги, методы и сроки оплаты

Расчетное время для перемещения платежей и Интернет-магазинов, из которых можно получить платежи, может различаться в зависимости от страны или региона и типа счета выплата. В этой статье описаны методы оплаты, поддерживаемые в разных странах и регионах.

В любой стране и регионе для оплаты используются платежи ACH/SEPA или банковские переводы. Некоторые страны и регионы также поддерживают PayPal в качестве метода оплаты.

> [!NOTE]
> Дополнительные сведения о том, когда корпорация Майкрософт перечисляет плату за коммерческое решение marketplace после выполнения транзакции, см. в разделе [Сведения о политике выплат](./payout-policy-details.md).

## <a name="number-of-days-for-payments-to-reach-payout-account"></a>Через сколько дней платежи поступают на счет для выплат

Как правило, мы отправляем любой платеж, подлежащий оплате в данном месяце, до 15-го числа этого месяца. Тем не менее, для поступления платежа на ваш счет может потребоваться дополнительное время. Количество дней зависит от метода оплаты, используемого для вашей учетной записи, как описано ниже.

| Способ оплаты | Приблизительное число дней для зачисления на счет |
| --- | --- |
| PayPal | 1 рабочий день |
| ACH/SEPA | 2–3 рабочих дня |
| Банковский перевод | 7–10 рабочих дней |
| | |

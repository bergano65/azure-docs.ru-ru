# <a name="create-azure-marketplace-and-appsource-offers"></a>Создание предложений Azure Marketplace и AppSource

Одной из основных целей Портала Cloud Partner является предоставление издателям возможностей создавать и публиковать предложения в Microsoft Azure Marketplace и AppSource.  Эта операция всегда начинается с выбора необходимого типа предложения в меню [New offer menu](../portal-tour/cpp-new-offer-menu.md) (Меню новых предложений).  После этого откроется соответствующая страница **Новое предложение** для этого типа предложения.  На следующем рисунке показана страница по умолчанию **Новое предложение** для типа "приложение Azure".

![Страница по умолчанию для нового предложения](./media/new-offer-page.png)

В горизонтальной строке меню, отображаемой в верхней части этой страницы, есть две вкладки. 
- **Редактор** — позволяет вводить сведения и передавать ресурсы для нового экземпляра предложения.  По умолчанию отображается эта вкладка.
- **Состояние** — предоставляет сведения о состоянии публикации, а также список всех проблем, выявленных различными проверками. 

При создании предложения для ввода сведений используется вкладка **Редактор**. 

## <a name="editing-operations"></a>Операции редактирования

На горизонтальной панели инструментов, расположенной над областью ввода данных, находятся следующие кнопки.

|   Кнопка    |   Назначение                                                          |
|   ------    |  --------                                                          |
| **Сохранить**    | Сохраняет все последние изменения ввода данных.  Необходимо вручную сохранять изменения, прежде чем переходить на другую страницу, иначе ваши изменения будут утрачены. | 
| **Отменить** | Отменяет последние изменения ввода данных (с момента последнего сохранения).             |
| **Сравнить** | Сравнивает состояние текущего предложения с опубликованным.  Доступна только после успешной публикации предложения.  |
| **Опубликовать** | Начинает процесс публикации предложения.                       |
| **Удалить**  | Удаляет созданное, но не опубликованное предложение. |
|   |   |


## <a name="editing-tabs"></a>Вкладки редактирования

При создании предложения указываются обязательные и необязательные данные на каждой вкладке, расположенной в левом вертикальном столбце на странице **Новое предложение**.  Стандартные элементы управления пользовательского интерфейса, такие как текстовые поля, раскрывающееся меню и флажки, отображены для сбора данных.  Хотя коллекции вкладок редактирования зависят от типа предложения, ниже перечислены распространенные вкладки.

|      Имя вкладки       |   Назначение                                                            |
|      --------       |   -------                                                            |
| **Параметры предложения**  | Собирает сведения о предложении и издателе.                    |
| **Номера SKU**            | Определяет технические и бизнес-характеристики для каждой версии номера SKU вашего предложения. |
| **Тестовый выпуск**      | Для тех типов предложений, которые поддерживают эту необязательную функцию, эта вкладка определяет демонстрацию вашего предложения.  Дополнительные сведения см. в статье о [тестовом выпуске](../test-drive/what-is-test-drive.md).  |
| **Marketplace** или **Storefront** (Онлайн-магазин) | Собирает текстовые строки, документы и изображения, используемые для отображения предложений в магазинах marketplace. |
| **Поддержка**         | Собирает контактную информацию для клиентов, проектирования и технической поддержки.  |
|  |  |

Содержимое вкладок с одним и тем же названием может отличаться в различных типах предложений.  Характерные для каждого предложения сведения предоставлены в разделе Create offer (Создать предложение).

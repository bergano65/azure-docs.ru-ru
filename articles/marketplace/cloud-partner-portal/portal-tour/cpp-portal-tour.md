# <a name="cloud-partner-portal-tour"></a>Знакомство с Порталом Cloud Partner

В этой статье содержатся вводные сведения о [Портале Cloud Partner](https://cloudpartner.azure.com): его содержимом, механизмах навигации и функциональных возможностях.   На этом портале зарегистрированные партнеры могут создавать предложения и управлять имеющимися предложениями для [Azure Marketplace](https://azuremarketplace.microsoft.com) и [AppSource Marketplace](https://azuremarketplace.microsoft.com).  Портал Cloud Partner доступен только для зарегистрированных партнеров. Дополнительные сведения см. в статье [Регистрация в качестве издателя в облаке Marketplace](https://docs.microsoft.com/azure/marketplace/become-publisher).

## <a name="section-contents"></a>Содержание раздела

После объяснения механизмов [навигации](./cpp-portal-navigation.md) портала мы рассмотрим основные записи в левой области навигации в соответствующих статьях: 
- Меню [Новое предложение](./cpp-new-offer-menu.md) позволяет пользователям создавать предложения выбранного типа.
- Страница [Все предложения](./cpp-all-offers-page.md) позволяет партнерам просматривать отправленные предложения и некоторые важнейшие характеристики. 
- Страница [Утверждения](./cpp-approvals-page.md) позволяет партнерам просматривать все одобренные на текущий момент предложения.
- На странице [Все издатели](./cpp-all-publishers-page.md) приведен список всех зарегистрированных на текущий момент облачных издателей.
- На странице [Пользователи](./cpp-users-page.md) приведен список всех учетных записей пользователей, связанных на текущий момент с зарегистрированными облачными партнерами.
- На странице [Журнал](./cpp-history-page.md) приведены все события публикации и изменения предложений по всем издателям. 
- Страница [Аналитические сведения](./cpp-insights-page.md) содержит различные панели мониторинга, которые относятся к компоненту портала *Seller Insights*.

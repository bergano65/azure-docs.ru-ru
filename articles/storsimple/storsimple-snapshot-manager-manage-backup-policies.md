# <a name="use-storsimple-snapshot-manager-to-create-and-manage-backup-policies"></a>Создание политик архивации и управление ими с помощью диспетчера моментальных снимков StorSimple
## <a name="overview"></a>Обзор
Политика архивации создает расписание архивации данных томов. Данные могут сохраняться как локально, так и в облаке. Создавая политику архивации, вы также можете указать политику хранения. (Вы можете сохранить максимум 64 снимков.) Для получения дополнительной информации о политике резервного копирования см. [Типы резервного копирования](storsimple-what-is-snapshot-manager.md#backup-types-and-backup-policies) в [серии StorSimple 8000: гибридное облачное решение.](storsimple-overview.md)

В этом учебнике объясняется, как выполнить такие задачи:

* создание политики архивации;
* внесение изменений в политику архивации;
* Удаление политики архивации

## <a name="create-a-backup-policy"></a>создание политики архивации;
Выполните указанные ниже действия, чтобы создать политику архивации.

#### <a name="to-create-a-backup-policy"></a>Создание политики архивации
1. Щелкните соответствующий значок на рабочем столе, чтобы запустить диспетчер моментальных снимков StorSimple.
2. На панели **Область** щелкните правой кнопкой мыши пункт **Политики резервного копирования**, а затем — **Создать политику резервного копирования**.

    ![создание политики архивации;](./media/storsimple-snapshot-manager-manage-backup-policies/HCS_SSM_Create_BU_policy.png)

    Откроется диалоговое окно **Создание политики**.

    ![Создание политики — вкладка "Общие"](./media/storsimple-snapshot-manager-manage-backup-policies/HCS_SSM_Create_policy_general.png)
3. На вкладке **Общие** введите следующие сведения.

   1. В поле **Имя** введите имя политики.
   2. В поле **Группа томов** укажите имя группы томов, связанной с политикой.
   3. Выберите пункт **Локальный моментальный снимок** или **Облачный моментальный снимок**.
   4. Выберите число сохраняемых моментальных снимков. Если вы выберете **Все**, будут сохранены 64 моментальных снимка (максимум).
4. Откройте вкладку **Расписание**.

    ![Создание политики — вкладка "Расписание"](./media/storsimple-snapshot-manager-manage-backup-policies/HCS_SSM_Create_policy_schedule.png)
5. На вкладке **Расписание** введите следующие сведения.

   1. Установите флажок **Включить**, чтобы запланировать резервное копирование.
   2. В разделе **Параметры** выберите **Один раз**, **Ежедневно**, **Еженедельно** или **Ежемесячно**.
   3. В поле **Начало** щелкните значок календаря и выберите дату начала.
   4. В разделе **Расширенные параметры** можно задать дополнительное расписание повторения и дату окончания.
   5. Нажмите кнопку **ОК**.

После создания политики резервного копирования на панели **Результаты** появятся следующие сведения.

* **Имя** — имя политики резервного копирования.
* **Тип** — локальный моментальный снимок или облачный моментальный снимок.
* **Группа томов** — группа томов, связанная с политикой.
* **Сохранение** — количество сохраняемых моментальных снимков (максимум 64).
* **Создано** — дата создания политики.
* **Включено** — указывает, активирована ли политика в настоящий момент. Значение **True** означает, что политика активирована, а значение **False** — что она отключена.

## <a name="edit-a-backup-policy"></a>внесение изменений в политику архивации;
Выполните указанные ниже действия, чтобы внести изменения в имеющуюся политику архивации.

#### <a name="to-edit-a-backup-policy"></a>Внесение изменений в политику архивации
1. Щелкните соответствующий значок на рабочем столе, чтобы запустить диспетчер моментальных снимков StorSimple.
2. На панели **Область** щелкните узел **Политики резервного копирования**. Все политики резервного копирования будут показаны на панели **Результаты**.
3. Щелкните правой кнопкой мыши политику, которую нужно изменить, а затем нажмите кнопку **Изменить**.

    ![внесение изменений в политику архивации;](./media/storsimple-snapshot-manager-manage-backup-policies/HCS_SSM_Edit_BU_policy.png)
4. При появлении окна **Создание политики** сделайте нужные изменения, затем нажмите кнопку **ОК**.

## <a name="delete-a-backup-policy"></a>Удаление политики архивации
Выполните указанные ниже действия, чтобы удалить политику архивации.

#### <a name="to-delete-a-backup-policy"></a>Удаление политики архивации
1. Щелкните соответствующий значок на рабочем столе, чтобы запустить диспетчер моментальных снимков StorSimple.
2. На панели **Область** щелкните узел **Политики резервного копирования**. Все политики резервного копирования будут показаны на панели **Результаты**.
3. Щелкните правой кнопкой мыши политику резервного копирования, которую нужно удалить, и выберите **Удалить**.
4. В подтверждающем сообщении нажмите кнопку **Да**.

    ![Подтверждение удаления политики архивации](./media/storsimple-snapshot-manager-manage-backup-policies/HCS_SSM_Delete_BU_policy.png)

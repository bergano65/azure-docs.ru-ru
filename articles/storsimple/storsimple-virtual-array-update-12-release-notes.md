# <a name="storsimple-virtual-array-update-12-release-notes"></a>Заметки о выпуске обновления 1,2 для виртуального массива StorSimple

В следующих заметках о выпуске перечислены важные нерешенные и устраненные проблемы, связанные с обновлениями для виртуального массива Microsoft Azure StorSimple.

Заметки о выпуске постоянно обновляются. По мере обнаружения новых критических проблем добавляется информация о способах их устранения. Перед развертыванием виртуального массива StorSimple внимательно изучите информацию, содержащуюся в заметках о выпуске.

Обновление 1,2 соответствует версии программного обеспечения **10.0.10311.0**.

> [!IMPORTANT]
> - Обновления нарушают работу системы и приводят к перезагрузке устройства. Если выполняются какие-либо операции ввода-вывода, то устройство находится в состоянии простоя. Подробные инструкции по пакетам, используемым для применения этого обновления, см. в [статье Загрузка обновления 1,2](#download-update-12).
> - Обновление 1,2 доступно через портал Azure только в том случае, если на устройстве выполняется обновление 1,0 или 1,1.

## <a name="whats-new-in-update-12"></a>Новые возможности в обновлении 1.2

Это обновление содержит следующие исправления ошибок.

- Улучшена устойчивость при обработке удаленных файлов.
- Улучшена обработка исключений в пути запуска кода, что приводит к снижению сбоев в резервных копиях, восстановлении, облачных операциях чтения и автоматизированной реорганизации пространства.

## <a name="download-update-12"></a>Загрузка обновления 1,2

Чтобы скачать это обновление, перейдите на сервер [каталога центр обновления Майкрософт](https://www.catalog.update.microsoft.com/Home.aspx) и скачайте пакет KB4502035. Этот пакет содержит следующие пакеты:

 - **KB4493446** , содержащий накопительные обновления Windows для 2012 R2 вплоть до 2019 апреля. Дополнительные сведения о том, что входит в эту свертку, см. в [ежемесячном сводном пакете безопасности](https://support.microsoft.com/help/4493446/windows-8-1-update-kb4493446).
 - **KB3011067** , который является центр обновления Майкрософт автономным файлом пакета ВИНДОВСС-KB3011067-x64. Этот файл используется для обновления программного обеспечения на устройстве.

Скачайте KB4502035 и следуйте этим инструкциям, чтобы [применить обновление через локальный веб-интерфейс](storsimple-virtual-array-install-update-11.md#use-the-local-web-ui).

## <a name="issues-fixed-in-update-12"></a>Проблемы, устраненные в обновлении 1.2

В таблице ниже содержится список проблем, устраненных в этом выпуске.

| Нет. | Функция | Проблема |
| --- | --- | --- |
| 1 |Удаление| В предыдущих версиях программного обеспечения возникла ошибка, когда использование устройства не изменилось даже при удалении файлов. Эта проблема исправлена в этой версии. При обработке удаленных файлов путь к коду многоуровневого кода был создан более устойчивым.|
| 2 |Обработка исключений| В предыдущих версиях программного обеспечения возникла проблема после перезагрузки системы, которая потенциально может привести к сбоям при резервном копировании, восстановлении, чтении из облака и автоматизированной реорганизации пространства. В этом выпуске содержатся изменения в способе обработки исключений в пути запуска.|

## <a name="known-issues-in-update-12"></a>Известные проблемы в обновлении 1.2

Новые проблемы не были выпущены в обновлении 1,2. Все проблемы, отмеченные в выпуске, переносятся из предыдущих выпусков. Чтобы просмотреть сводку известных проблем, связанных с предыдущими выпусками, перейдите к разделу [Известные проблемы в обновлении 1,1](storsimple-virtual-array-update-11-release-notes.md#known-issues-in-update-11).

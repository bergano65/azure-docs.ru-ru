# <a name="http2-support-in-azure-front-door"></a>Поддержка HTTP/2 в передней дверце Azure

В настоящее время поддержка HTTP/2 активна для всех конфигураций передней дверцы Azure. Никаких действий со стороны пользователей не требуется.

HTTP/2 — это основная редакция HTTP/1.1, которая обеспечивает более быструю веб-производительность за счет сокращения времени отклика. HTTP/2 выполняется с помощью знакомых методов HTTP, кодов состояния и семантики для улучшения взаимодействия с пользователем. Хотя HTTP/2 разработана для работы с HTTP и HTTPS, многие клиентские веб-браузеры поддерживают HTTP/2 только по протоколу TLS.

> [!NOTE]
> Поддержка протокола HTTP/2 доступна только для запросов от клиентов к передней дверце. Обмен данными между передней дверцей и серверной частью серверного пула происходит по протоколу HTTP/1.1. 

### <a name="http2-benefits"></a>Преимущества HTTP/2

Преимущества HTTP/2 перечислены ниже.

*   **Мультиплексирование и параллелизм**

    При использовании HTTP 1.1 для выполнения нескольких запросов к ресурсам требуется несколько TCP-подключений, и с каждым подключением связаны определенные затраты на обеспечение производительности. HTTP/2 позволяет запрашивать несколько ресурсов через одно TCP-подключение.

*   **Сжатие заголовка**

    Благодаря сжатию HTTP-заголовков для обслуживаемых ресурсов значительно уменьшается время в сети.

*   **Зависимости потоков**

    Зависимости потоков позволяют клиенту указывать серверу, какие ресурсы имеют приоритет.


## <a name="http2-browser-support"></a>Поддержка HTTP/2 в браузерах

Во всех текущих версиях основных браузеров реализована поддержка HTTP/2. Неподдерживаемые браузеры автоматически переходят на HTTP/1.1.

|-------------|------------|
|Microsoft Edge| 12|
|Google Chrome| 43|
|Mozilla Firefox;| 38|
|Opera| 32|
|Safari| 9|

# <a name="technical-specifications-and-compliance-for-azure-stack-edge-pro-with-gpu"></a>Технические спецификации и соответствие требованиям для Azure Stack пограничных Pro с помощью GPU 

Аппаратные компоненты Azure Stack ребра Pro с интегрированным графическим процессором (GPU) соответствуют техническим спецификациям и нормативным стандартам, изложенным в этой статье. Технические спецификации описывают оборудование, единицы питания (блока питания), емкость хранилища, корпуса и стандарты окружающей среды.

## <a name="compute-and-memory-specifications"></a>Спецификации вычислений и памяти

Azure Stack пограничных устройств Pro имеет следующие спецификации для вычислений и памяти.

| Спецификация           | Значение                  |
|-------------------------|----------------------------|
| ЦП                     | 2 процессора Intel Xeon серебро 4214 (Cascade Lake)            |
| Память                  | 128 ГБ ОЗУ (8x16 ГБ)                     |


## <a name="compute-acceleration-specifications"></a>Спецификации ускорения вычислений

Графический процессор (GPU) включается на каждое устройство Azure Stack пограничной Pro, позволяющее выполнять сценарии Kubernetes, глубокого обучения и машинного обучения.

| Спецификация           | Значение                  |
|-------------------------|----------------------------|
| Графический процессор   | Один или два GPU nVidia T4 <br> Дополнительные сведения см. в разделе [NVIDIA T4](https://www.nvidia.com/en-us/data-center/tesla-t4/).| 


## <a name="power-supply-unit-specifications"></a>Спецификации источника питания

Устройство Azure Stack пограничной Pro имеет два блока питания 100-240 V (блока питания) с высокопроизводительными вентиляторами. Два таких источника питания обеспечивают конфигурацию с избыточным питанием. В случае сбоя источника питания устройство продолжает работать в обычном режиме на другом источнике питания до замены отказавшего модуля. В таблице ниже приведены технические спецификации модулей питания.

| Спецификация           | БП 750 ВТ                  |
|-------------------------|----------------------------|
| Максимальная выходная мощность    | 750 Вт                     |
| Частота               | 50/60 Гц                   |
| Выбор диапазона напряжения | Автоматический выбор напряжения: 100–240 В переменного тока |
| «Горячая» замена           | Да                        |


## <a name="network-interface-specifications"></a>Спецификации сетевых интерфейсов

Устройство Azure Stack пограничной Pro имеет шесть сетевых интерфейсов PORT1-PORT6.

| Спецификация           | Описание                 |
|-------------------------|----------------------------|
|  Сетевые интерфейсы    | **2 X 1 гигабитных** интерфейса — 1 интерфейс управления. порт 1 используется для начальной настройки и является статическим по умолчанию. После завершения начальной настройки можно использовать интерфейс для данных с любым IP-адресом. Однако при сбросе интерфейс возвращается к статическому IP-адресу. <br>Другой порт 2 пользовательского интерфейса можно настроить для передачи данных, а по умолчанию — DHCP. <br>**4 X 25 интерфейсов** : эти интерфейсы данных, порт 3 через порт 6, могут быть настроены пользователем как DHCP (по умолчанию) или статическими. Они также могут действовать как 10 интерфейсов GbE.  | 

Устройство Azure Stack пограничной Pro имеет следующее сетевое оборудование:

* **Пользовательский адаптер Microsoft QLogic Cavium 25G NDC** — порт 1 через порт 4.
* **Канальный сетевой адаптер Mellanox ConnectX-4 с двумя портами 25G** — порт 5 и 6.

Ниже приведены сведения о карте Mellanox:

| Номер части устройства (R640) | MCX4121A — АКАТ  |
| ПУСТОЙ PSID (R640)           | MT_2420110034                         |

Полный список поддерживаемых кабелей, коммутаторов и приемопередатчиков для этих сетевых карт см. на следующих страницах.

- [Таблица взаимодействия адаптера QLogic Cavium 25G NDC](https://www.marvell.com/documents/xalflardzafh32cfvi0z/).

## <a name="storage-specifications"></a>Спецификации хранилища

Общая используемая емкость для устройства составляет примерно 8,28 ТБ. В следующей таблице перечислены объемы хранилища устройства.

|    Число твердотельных накопителей NVMe                     |    5                  |
|    Емкость диска SSD                       |    240 ГБ             |
|    Одноядерная емкость SSD NVMe                |    1,6 TБ             |
|    Общая емкость                          |    8,28 ТБ            |
|    Общая доступная емкость*                  |    ~ 7,95 ТБ          |
|    Контроллер SAS                          |    HBA330 12 Гбит/с     |

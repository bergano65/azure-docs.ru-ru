# <a name="azure-stack-edge-pro-r-technical-specifications"></a>Технические спецификации R для Azure Stack пограничных устройств

Аппаратные компоненты устройства R для Azure Stack ребра соответствуют техническим спецификациям, изложенным в этой статье. В технических спецификациях описываются источники питания, габаритные размеры, емкость и стандарты охраны окружающей среды.


## <a name="compute-memory-specifications"></a>Спецификации вычислительных компонентов и запоминающих устройств

Устройство Azure Stack ребра Pro R имеет следующие спецификации для вычислений и памяти:

| Спецификация       | Значение                  |
|---------------------|------------------------|
| ЦП    | 2 X 10 ядер ЦП, Intel Xeon серебро 4114 |
| Память              | 256 ГБ ОЗУ (2666 MT/с)     |


## <a name="compute-acceleration-specifications"></a>Спецификации ускорения вычислений

Графический процессор (GPU) включен на каждое устройство, позволяющее Kubernetes, глубокое обучение и сценарии машинного обучения.

| Спецификация           | Значение                  |
|-------------------------|----------------------------|
| Графический процессор   | Один графический процессор nVidia T4 <br> Дополнительные сведения см. в разделе [NVIDIA T4](https://www.nvidia.com/en-us/data-center/tesla-t4/).| 

## <a name="power-supply-unit-specifications"></a>Спецификации источника питания

Устройство Azure Stack пограничной Pro R имеет два блока питания 100-240 V (блока питания) с высокопроизводительными вентиляторами. Два таких источника питания обеспечивают конфигурацию с избыточным питанием. В случае сбоя источника питания устройство продолжает работать в обычном режиме на другом источнике питания до замены отказавшего модуля. В таблице ниже приведены технические спецификации модулей питания.

| Спецификация           | 550 ВТ                  |
|-------------------------|----------------------------|
| Максимальная выходная мощность    | 550 Вт                      |
| Тепловыделение (максимум)                   | 2891 БТУ/ч                |
| Частота               | 50/60 Гц                   |
| Выбор диапазона напряжения | Автоматический диапазон: 115-230 V AC |
| «Горячая» замена           | Да                        |

## <a name="network-specifications"></a>Спецификации сети

Устройство Azure Stack пограничной Pro R имеет четыре сетевых интерфейса PORT1-PORT4. 


|Спецификация  |Описание                              |
|----------------------|----------------------------------|
|Сетевые интерфейсы    |**2 x 1 GbE RJ45** <br> Порт 1 используется в качестве интерфейса управления для начальной настройки и является статическим по умолчанию. После завершения начальной настройки можно использовать интерфейс для данных с любым IP-адресом. Однако при сбросе интерфейс возвращается к статическому IP-адресу. <br>Другой порт 2 пользовательского интерфейса можно настроить для передачи данных, а по умолчанию — DHCP.     |
|Сетевые интерфейсы    |**2 x 25 GbE SFP28** <br> Эти интерфейсы данных порт 3 и порт 4 можно настроить как DHCP (по умолчанию) или статический.            |

Устройство Azure Stack пограничной Pro R имеет следующее сетевое оборудование:

* **Сетевой адаптер Mellanox Dual-4 Channel** , порт 3 и порт 4.

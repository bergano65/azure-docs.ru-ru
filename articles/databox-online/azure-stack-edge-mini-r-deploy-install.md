# <a name="tutorial-install-azure-stack-edge-mini-r"></a>Руководство по Установка Azure Stack Edge Mini R

Это руководство содержит инструкции по установке Azure Stack Edge Mini R на физическое устройство. В процедуру установки входит подключение кабелей к устройству.

Установка может занять примерно 30 минут.

В этом руководстве описано следующее.

> [!div class="checklist"]
> * Изучение устройства
> * Подключение кабелей к устройству

## <a name="prerequisites"></a>Предварительные требования

Ниже приведены предварительные требования для установки физического устройства.

### <a name="for-the-azure-stack-edge-resource"></a>Для ресурса Azure Stack Edge

Перед тем как начать, убедитесь в следующем.

* Выполнены все инструкции из руководства по [подготовке к развертыванию Azure Stack Edge Mini R](azure-stack-edge-mini-r-deploy-prep.md).
    * Создан ресурс Azure Stack Edge для развертывания устройства.
    * Создан ключ активации, чтобы активировать устройство с помощью ресурса Azure Stack Edge.

 
### <a name="for-the-azure-stack-edge-mini-r-physical-device"></a>Для физического устройства Azure Stack Edge Mini R

Перед развертыванием устройства:

- Убедитесь, что устройство надежно установлено на плоской, стабильной и ровной рабочей поверхности.
- Убедитесь, что в расположении, в котором планируется выполнить настройку, есть следующее:
    - либо стандартное питание переменным током от независимого источника;
    - либо стоечный блок распределения питания (PDU). 
    

### <a name="for-the-network-in-the-datacenter"></a>Сеть в центре обработки данных

Перед началом работы

- Изучите сетевые требования к развертыванию Azure Stack Edge Mini R и настройте сеть центра обработки данных в соответствии с этими требованиями. Дополнительные сведения см. в разделе [Требования к сети Azure Stack Edge](azure-stack-edge-mini-r-system-requirements.md#networking-port-requirements).

- Убедитесь, что интернет-канал имеет пропускную способность не ниже 20 Мбит/с для оптимального функционирования устройства. <!-- engg TBC -->


## <a name="inspect-the-device"></a>Изучение устройства

Данное устройство поставляется единым блоком. Выполните следующие действия для распаковки устройства.

1. Расположите коробку на ровной плоской поверхности.
2. Проверьте, не повреждена ли упаковка устройства. Откройте упаковку и проверьте само устройство. Если упаковка устройства имеет следы повреждений, обратитесь в службу поддержки Майкрософт, которая поможет в дальнейшей оценке состояния устройства.
3. После вскрытия упаковки проверьте комплектацию устройства:
    - одно переносное устройство Azure Stack Edge Mini R с установленными боковыми накладками;
    - один аккумулятор с задней крышкой, присоединенный к устройству; 
    - один шнур питания для подключения аккумулятора к источнику питания. 

Если вы не получили все элементы из перечисленных в этом руководстве, обратитесь в службу технической поддержки Azure Stack Edge. Следующим шагом будет подключение кабелей к устройству.


## <a name="cable-the-device"></a>Подключение кабелей к устройству

Ниже описано, как подсоединить кабель питания и сетевой кабель к устройству Azure Stack Edge.

Перед подключением кабелей к устройству убедитесь, что у вас есть следующее:

- физическое устройство Azure Stack Edge Mini R на месте планируемой установки;
- один кабель питания;
- как минимум один сетевой кабель 1 GbE RJ-45 для подключения к интерфейсу управления (на устройстве есть два сетевых интерфейса 1 GbE: для управления и для данных);
- один медный кабель стандарта 10 GbE SPF+ для каждого настраиваемого сетевого интерфейса; хотя бы один сетевой интерфейс данных (PORT 3 или PORT 4) должен быть подключен к Интернету и к сети Azure;  
- возможность использовать один блок распределения питания.

> [!NOTE]
> - Если вы подключаете только один сетевой интерфейс данных, для отправки данных в Azure мы рекомендуем использовать сетевой интерфейс 10 GbE, например PORT 3 или PORT 4. 
> - Для достижения оптимальной производительности и обработки больших объемов данных рекомендуем подключить все порты данных.
> - Устройство Azure Stack Edge необходимо подключить к сети центра обработки данных таким образом, чтобы оно могло принимать данные с исходных серверов данных. <!-- engg TBC -->

На устройстве Azure Stack Edge:

- На передней панели есть салазки SSD. 

    - В этот слот к устройству подключен 1 диск SSD. 
    - Кроме того, устройство оснащено картой CFx, которая служит хранилищем для диска операционной системы.
    
- На передней панели есть сетевые интерфейсы и порт Wi-Fi:

    - 2 сетевых интерфейса 1 GbE RJ 45. В локальном пользовательском интерфейсе устройства они отображаются как PORT 1 и PORT 2.
    - 2 сетевых интерфейса 10 GbE SPF+. В локальном пользовательском интерфейсе устройства они отображаются как PORT 3 и PORT 4. 
    - Один порт Wi-Fi с подключенным к нему трансивером Wi-Fi.

- Кроме того, на передней панели есть кнопка питания. 

- На задней панели размещены аккумулятор и крышка, которые установлены на устройстве. 


Выполните следующие действия, чтобы подключить к устройству питание и сеть.

1. Определите назначение компонентов на передней панели устройства для подключения сети и накопителей.

    ![Интерфейсы для подключения сети и накопителей на устройстве](./media/azure-stack-edge-mini-r-deploy-install/ports-front-plane.png)

2. Найдите кнопку питания в нижнем углу слева на передней панели устройства. 

    ![Передняя панель устройства с кнопкой питания на устройстве](./media/azure-stack-edge-mini-r-deploy-install/device-power-button.png)

3. Аккумулятор подключен к задней панели устройства. Найдите вторую кнопку питания, расположенную на аккумуляторе. 

    ![Передняя панель устройства с кнопкой питания на аккумуляторе](./media/azure-stack-edge-mini-r-deploy-install/battery-power-button.png)


    Подключите один конец шнура питания к аккумулятору, а другой к розетке. 

    ![Подключение кабеля питания](./media/azure-stack-edge-mini-r-deploy-install/power-cord-connector-1.png) 

    При работе только от аккумулятора (без подключения к источнику питания) оба выключателя питания (на передней панели и на аккумуляторе) должны находиться во включенном положении. Когда аккумулятор подключен к источнику питания, должна быть включена только кнопка питания на передней панели устройства. 

4. Включите узлы устройства. Для этого нажмите кнопку питания на передней панели. 
    
    > [!NOTE]
    > Чтобы включить или отключить питание устройства, нажмите черную кнопку, расположенную над кнопкой питания. Затем переведите кнопку питания в нужное положение. 

5. Для настройки Wi-Fi на устройстве используйте следующую схему подключения кабелей:

    ![Подключение кабелей для Wi-Fi](./media/azure-stack-edge-mini-r-deploy-install/wireless-cabled.png)  

    - Подключите сетевой интерфейс 1 GbE PORT 1 к компьютеру, используемому для настройки физического устройства. PORT 1 — это выделенный интерфейс управления.


    Если для устройства используется проводное подключение, используйте следующую схему:
     
    ![Подключение кабелей для проводной сети](./media/azure-stack-edge-mini-r-deploy-install/wired-cabled.png)     
    - Подключите сетевой интерфейс 1 GbE PORT 1 к компьютеру, используемому для настройки физического устройства. PORT 1 — это выделенный интерфейс управления.
    - Подключите к сети центра обработки данных или напрямую к Интернету порты PORT 2, PORT 3 и (или) PORT 4 в любом сочетании.
    
        - Для порта PORT 2 используйте сетевой кабель RJ-45.
        - Для сетевых интерфейсов 10 GbE используйте медные кабели SFP+.

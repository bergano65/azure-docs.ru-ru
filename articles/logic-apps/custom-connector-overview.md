# <a name="custom-connectors-in-logic-apps"></a>Пользовательские соединители в Logic Apps

Вы можете создавать рабочие процессы и приложения с помощью [Azure Logic Apps](https://azure.microsoft.com/services/logic-apps), [Power Automate](https://flow.microsoft.com) и [Power Apps](https://powerapps.microsoft.com) без написания кода. Чтобы помочь вам интегрировать приложения, данные и бизнес-процессы, эти службы предлагают [более 200 соединителей](/connectors/). Они подходят для служб и продуктов Майкрософт, таких как Azure и SQL Server, а также сторонних служб, таких как GitHub, Salesforce, Twitter и т. д.

Тем не менее, иногда требуется вызвать API, службы или системы, которые не предоставляются в качестве готовых соединителей. Для поддержки более специализированных сценариев можно создать *пользовательские соединители*, которые будут содержать собственные триггеры и действия. На сайте с документацией о соединителях имеются базовые и расширенные руководства по пользовательским соединителям. Начните с [обзора пользовательских соединителей](/connectors/custom-connectors/). * [Create a custom connector from a Postman collection](/connectors/custom-connectors/define-postman-collection) (Создание пользовательского соединителя из коллекции Postman)

* [Use a custom connector from a logic app](/connectors/custom-connectors/use-custom-connector-logic-apps)(Применение пользовательского соединителя из приложения логики)

* [Совместное использование пользовательского соединителя в организации](/connectors/custom-connectors/share)

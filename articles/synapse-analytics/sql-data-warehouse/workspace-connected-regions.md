# <a name="enabling-synapse-workspace-features-on-a-dedicated-sql-pool-formerly-sql-dw-in-your-region"></a>Включение функций рабочей области синапсе в выделенном пуле SQL (ранее — в хранилище данных SQL) в вашем регионе.

## <a name="regions-not-supported"></a>Регионы не поддерживаются 
В следующей таблице представлена сводка по регионам, где возможность включения функций рабочей области синапсе в существующий выделенный пул SQL (ранее — хранилище данных SQL) не поддерживается. Следите за обновлениями, чтобы узнать, когда ваш регион станет доступен.

\* указывает, что определенное расписание для региона в настоящее время недоступно.

| Регион | Доступность рабочей области синапсе |
|:--- |:--- |:--- |
| Центральная Австралия 1 |\* |
| Центральная Австралия 2 |\* |
| Восточный Китай 2 |31 марта 2021 г. |
| Северный Китай 2 |31 марта 2021 г. |
| Южная Индия |31 марта 2021 г. |
| Западная Индия |31 марта 2021 г. |
| Республика Корея, южный регион |31 марта 2021 г. |
| Западная Швейцария |31 марта 2021 г. |
| Северная часть ОАЭ; |31 марта 2021 г. |
 
 
> [!NOTE]
> Список перечисленных выше регионов относится только к регионам, в которых включение функций рабочей области синапсе в настоящее время недоступно.

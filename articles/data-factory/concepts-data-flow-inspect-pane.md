# <a name="azure-data-factory-mapping-data-flow-transformation-inspect-tab"></a>Вкладка проверки преобразования сопоставления потоков данных Фабрики данных Azure

[!INCLUDE [notes](../../includes/data-factory-data-flow-preview.md)]

![Область проверки](media/data-flow/agg3.png "Область проверки")

Область проверки предоставляет представление в метаданных потока данных, который вы преобразуете. Вы сможете увидеть количество столбцов, измененные и добавленные столбцы, типы данных, порядок и ссылки на столбцы. "Проверка" — это представление метаданных только для чтения. Чтобы увидеть метаданные в области проверки вам не нужно включать режим "Отладка".

Во время изменения формы ваших данных с помощью преобразований, вы увидите, как метаданные изменяют поток с помощью области проверки. Метаданные не будут отображаться в области проверки, если в преобразовании источника не определена схема. Отсутствие метаданных часто встречается в сценариях смещения схемы.

![Предварительный просмотр данных](media/data-flow/datapreview.png "Предварительный просмотр данных")

"Предварительный просмотр данных" — это область, которая обеспечивает представление данных только для чтения в процессе их преобразования. Вы можете просмотреть выходные данные преобразования и выражения, чтобы проверить поток данных. Чтобы увидеть предварительные версии данных необходимо включить режим "Отладка". При нажатии на столбцы в сетке предварительного просмотра данных справа отобразится следующая панель. На всплывающей панели отобразится статистика профиля каждого выбранного столбца.

![Диаграмма "Предварительный просмотр данных"](media/data-flow/chart.png "Диаграмма \"Предварительный просмотр данных\"")

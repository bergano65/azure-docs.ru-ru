# <a name="azure-event-hubs---resource-manager-exceptions"></a>Концентраторы событий Azure — диспетчер ресурсов исключения
В этой статье перечислены исключения, созданные при взаимодействии с концентраторами событий Azure с помощью шаблонов Azure Resource Manager или прямых вызовов.

> [!IMPORTANT]
> Этот документ часто обновляется. Ознакомьтесь с обновлениями.

В следующих разделах приводятся различные исключения и ошибки, отображаемые в Azure Resource Manager.

## <a name="error-code-conflict"></a>Код ошибки: конфликт

| Код ошибки | Код ошибки | Сообщение об ошибке | Описание | Рекомендация |
| ---------- | ------------- | ------------- | ----------- | -------------- |
| Conflict | 40300 | Достигнуто или превышено максимальное число ресурсов типа EventHub. Фактически: #, максимально допустимое число: # | Пространство имен достигло своей [квоты](event-hubs-quotas.md) для числа концентраторов событий, которые он может содержать. | Удалите все неиспользуемые или внешние концентраторы событий из пространства имен или рассмотрите возможность обновления до [выделенного кластера](event-hubs-dedicated-overview.md). |
| Conflict | нет | Невозможно удалить конфигурацию аварийного восстановления (DR), так как выполняется репликация. Отработка отказа или прерывание связывания перед удалением конфигурации аварийного восстановления. | Выполняется [репликация GeoDR](event-hubs-geo-dr.md) , поэтому в данный момент невозможно удалить конфигурацию. | Чтобы разблокировать удаление конфигурации, дождитесь завершения репликации, запустите отработку отказа или разбейте пару GeoDR. |
| Conflict | нет | Сбой обновления пространства имен с конфликтом в серверной части. | В этом пространстве имен выполняется другая операция. | Дождитесь завершения текущей операции и повторите попытку. |

## <a name="error-code-429"></a>Код ошибки: 429

| Код ошибки | Код ошибки | Сообщение об ошибке | Описание | Рекомендация |
| ---------- | ------------- | ------------- | ----------- | -------------- |
| 429 | нет | Подготовка пространства имен в переходе | В этом пространстве имен выполняется другая операция. | Дождитесь завершения текущей операции и повторите попытку. |
| 429 | нет | Выполняется операция аварийного восстановления. | Операция [GeoDR](event-hubs-geo-dr.md) в настоящее время выполняется в этом пространстве имен или связывании. | Дождитесь завершения текущей операции GeoDR и повторите попытку. |

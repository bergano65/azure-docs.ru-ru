# <a name="language-support-for-computer-vision"></a>Языковая поддержка для Компьютерное зрение

Некоторые функции Компьютерное зрение поддерживают несколько языков. все функции, не упомянутые здесь, поддерживают только английский язык.

## <a name="text-recognition"></a>Распознавание текста

Компьютерное зрение может распознать текст на многих языках. В частности, API [OCR](https://westus.dev.cognitive.microsoft.com/docs/services/5adf991815e1060e6355ad44/operations/56f91f2e778daf14a499e1fc) поддерживает различные языки, тогда как API [чтения](https://westus.dev.cognitive.microsoft.com/docs/services/5adf991815e1060e6355ad44/operations/2afb498089f74080d7ef85eb) и API [Распознавание текста](https://westus.dev.cognitive.microsoft.com/docs/services/5adf991815e1060e6355ad44/operations/587f2c6a154055056008f200) поддерживают только английский язык. Дополнительные сведения об этих функциях и преимуществах каждого API см. в разделе [распознавание печатного и рукописного текста](concept-recognizing-text.md) .

OCR автоматически определяет язык входного материала, поэтому нет необходимости указывать код языка в вызове API. Однако коды языка всегда возвращаются в качестве значения `"language"` узла в ответе JSON.

## <a name="image-analysis"></a>Анализ изображений

# <a name="computer-vision-api-frequently-asked-questions"></a>Часто задаваемые вопросы об API компьютерного зрения

> [!TIP]
> Если вы не найдете ответ на свой вопрос в этом списке, попробуйте обратиться к сообществу разработчиков для API компьютерного зрения на сайте [StackOverflow](https://stackoverflow.com/questions/tagged/project-oxford+or+microsoft-cognitive) или связаться со [службой поддержки UserVoice](https://cognitive.uservoice.com/).

-----

**Вопрос**. *Можно ли обучить API компьютерного зрения использованию настраиваемых тегов?  Например, мне нужно передавать изображения различных пород кошек, чтобы обучить ИИ, а затем получать значение породы по запросу ИИ.*

**Ответ**. Эта функция сейчас недоступна. Тем не менее наши инженеры работают над тем, чтобы добавить ее в API компьютерного зрения.

-----

**Вопрос**. *Можно ли использовать API компьютерного зрения локально, без подключения к Интернету?*

**Ответ**. В настоящее время локальное решение отсутствует.

-----

**Вопрос**. *Можно ли использовать API компьютерного зрения для чтения номерных знаков?*

**Ответ**. API компьютерного зрения обеспечивает хорошее обнаружение и распознавание текста, но в настоящее время он не оптимизирован для номерных знаков. Мы постоянно прилагаем усилия, чтобы улучшить качество наших услуг. Мы добавили автоматическое распознавание текста номерных знаков в список запрашиваемых функций.

-----

**Вопрос**. *Какие типы поверхности поддерживаются для распознавания рукописного текста?*

**Ответ**. Эта технология работает с различными видами поверхностей, включая доски, технические документы и записки.

-----

**Вопрос**. *Какова продолжительность распознавания рукописного текста?*

**Ответ**. Продолжительность зависит от размера текста. Длинные тексты могут распознаваться в течение нескольких секунд. Таким образом после завершения операции распознавания рукописного текста может потребоваться подождать, прежде чем можно будет извлечь ее результаты с помощью операции получения результата распознавания рукописного текста.

-----

**Вопрос**. *Как технология распознавания рукописного текста обрабатывает текст, добавленный с помощью курсора в середину строки?*

**Ответ**. Операция распознавания рукописного текста возвращает такой текст в виде отдельной строки.

-----

**Вопрос**. *Как технология распознавания рукописного текста обрабатывает зачеркнутые слова или строки?*

**Ответ**. Если слова зачеркнуты несколькими линиями, чтобы их невозможно было распознать, операция распознавания рукописного текста не извлекает их. Но если слова зачеркнуты одной линией, то это зачеркивание считается помехой, и эти слова все равно извлекаются операцией распознавания рукописного текста.

-----

**Вопрос**. *Какие ориентации текста поддерживаются технологией распознавания рукописного текста?*

**Ответ**. Операция распознавания рукописного текста может принимать текст под углом примерно до 30–40 градусов.

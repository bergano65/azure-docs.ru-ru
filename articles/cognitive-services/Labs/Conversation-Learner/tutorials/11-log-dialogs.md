# <a name="how-to-log-dialogs-in-a-conversation-learner-model"></a>Запись диалогов в модели Conversation Learner

В этом руководстве содержатся сведения об использовании записываемых диалогов в целях более эффективного обучения моделей Conversation Learner из записанных взаимодействий с реальными пользователями.

## <a name="video"></a>Видео

[![Предварительная версия руководства по использованию записываемых диалогов](https://aka.ms/cl_Tutorial_v3_LogDialogs_Preview)](https://aka.ms/cl_Tutorial_v3_LogDialogs)

## <a name="requirements"></a>Требования
Для работы с этим руководством требуется запущенный бот обучения общего назначения.

    npm run tutorial-general

Также необходима модель погоды, созданная в предыдущих руководствах.

## <a name="details"></a>Сведения
Записываемые диалоги — это записанные журналы взаимодействия вашего бота с конечными пользователями. С их помощью вы можете исправить метки сущностей и выбранные действия, чтобы повысить производительность модели и системы в целом.

## <a name="steps"></a>Действия

В веб-интерфейсе щелкните Import Tutorials (Импорт учебников) и выберите модель Tutorial-11-LogDialogs.

Эта модель содержит одну сущность с именем city (город) и действия, предназначенные для ответов на вопросы о погоде в этом городе. Для обучения модели использовались два обучающих диалога, поэтому ожидаемая производительность является довольно низкой. Модель можно усовершенствовать за счет дополнительного обучения и взаимодействия с реальными пользователями.

### <a name="create-a-new-conversation"></a>Создание беседы

1. На левой панели щелкните "Записываемые диалоги" и нажмите кнопку "Создать записываемый диалог".
2. На панели чата, где которой отображается надпись "Введите текст сообщения...", введите текст Austin weather forecast (Прогноз погоды в Остине).
3. Нажмите кнопку Done Testing (Тестирование завершено).
4. В списке выберите записываемый диалог Austin weather forecast (Прогноз погоды в Остине).
5. На панели чата щелкните фразу Austin weather forecast (Прогноз погоды в Остине).
6. В списке сущностей выберите Austin (Остин), затем — city (город).
7. Нажмите кнопку Submit Changes (Отправить изменения).
    - Это изменение в значении сущности приведет к последующим изменениям в беседе, поскольку в памяти есть новые значения сущности. Дальнейшие действия, скорее всего, стали недействительными. Особенно те, в которых использовалась сущность city (город).
8. На панели чата щелкните фразу Which city? (Какой город?).
9. Выберите ответ The weather in Austin is probably sunny (Погода в Остине, скорее всего, солнечная).
10. Нажмите кнопку Save As Train Dialog (Сохранить как обучающий диалог).
    - Сразу же запустится обучение.

![](../media/T11_logdialog.png)

Последнее замечание. В зависимости от потребностей бизнеса функцию записи беседы можно отключить. Для этого в разделе параметров нужно снять флажок Log Conversations (Записывать беседы).

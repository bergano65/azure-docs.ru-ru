# <a name="how-to-use-cards-part-1-of-2"></a>Использование карт (часть 1 из 2)

В этом руководстве показано, как добавить и использовать простую карту в боте.

> [!NOTE]
> Сейчас для работы Conversation Learner файлы с определениями карт должны размещаться в каталоге с именем cards, созданном в каталоге, из которого запущен бот.

## <a name="requirements"></a>Требования
Для работы с этим руководством требуется запущенный бот обучения общего назначения.

    npm run tutorial-general

## <a name="details"></a>Сведения

Карты — это элементы пользовательского интерфейса, которые позволяют пользователю выбрать вариант ответа в диалоге. 

### <a name="open-the-demo"></a>Запуск демонстрационного примера

В веб-интерфейсе щелкните "Импорт учебников" и выберите модель Tutorial-15-Cards.

### <a name="the-card"></a>Карта

Определение карты находится в следующем расположении: C:\<каталог_установки\>\src\cards\prompt.json.

Система ищет определения карт в указанном каталоге cards.

![](../media/tutorial13_prompt.PNG)

> [!NOTE]
> Обратите внимание на тип текста TextBlock и заполнитель {{question}} в текстовом поле.
> Доступны две кнопки отправки и текст, который передается каждой из них.

### <a name="actions"></a>Действия

Мы создали три действия. Как показано ниже, первое действие представляет собой карту.

![](../media/tutorial13_actions.PNG)

Давайте посмотрим, как создается этот тип действия.

![](../media/tutorial13_cardaction.PNG)

> [!NOTE]
> Карта содержит три различных параметра: вопрос, кнопку 1 и кнопку 2. Эти элементы представляют собой ссылки на шаблон в карте, куда вы вводите вопрос и соответствующие ответы. Также можно ссылаться на сущности или на сочетание текста и сущностей и использовать их.

Значок в виде глаза показывает, как выглядит карта.

### <a name="practicing-creating-card-actions"></a>Практическое создание карт

1. На левой панели щелкните "Действия", а затем нажмите кнопку "Создать действие".
2. Выберите CARD (Карта) в поле "Тип действия".
3. В списке "Шаблон" выберите prompt (запрос).
4. В поле "вопрос" введите Go left or right (Перейти влево или вправо).
5. В поле "кнопка 1" введите left (влево).
6. В поле "кнопка 2" введите right (вправо).
7. Нажмите кнопку "Отмена".

### <a name="train-dialog-using-an-adaptive-card"></a>Обучающий диалог с использованием адаптивной карты

1. На левой панели щелкните "Обучающие диалоги" и нажмите кнопку "Создать обучающий диалог".
2. На панели чата, на которой отображается надпись "Введите текст сообщения...", введите hi (привет).
3. Нажмите кнопку "Оценить действия".
4. Выберите ответ prompt: question: Go left or right? (Запрос: вопрос: перейти влево или вправо?)
    - Для предварительного просмотра карты щелкните значок с изображением глаза.
5. На панели чата нажмите кнопку Left (Влево) в созданном запросе.
6. Нажмите кнопку "Оценить действия".
7. Выберите ответ Left (Влево).
8. Нажмите кнопку "Сохранить".
9. Выберите ответ prompt: question: Go left or right? (Запрос: вопрос: перейти влево или вправо?)
10. На панели чата нажмите кнопку Right (Вправо) в созданном запросе.
11. Нажмите кнопку "Оценить действия".
12. Выберите ответ Right (Вправо).

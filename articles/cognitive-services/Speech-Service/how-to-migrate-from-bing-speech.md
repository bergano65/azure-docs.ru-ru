# <a name="migrate-from-bing-speech-to-the-speech-service"></a>Перенос из службы "Распознавание речи Bing" в службу "Речь"

В этой статье показано, как выполнить миграцию приложений из API распознавания речи Bing в службу "Речь".

В этой статье приведены различия между API-интерфейсы распознавания речи Bing и службы распознавания речи и предлагает рекомендации по переносу приложений. Ключ подписки API распознавания речи Bing не будет работать со службой речи; Вам потребуется новую подписку службы распознавания речи.

Один ключ подписки служб речи предоставляет доступ к следующие функции. Каждый тарифицируется отдельно, поэтому плата взимается только за функции, которые вы используете.

* [Преобразование речи в текст](speech-to-text.md)
* [Настраиваемое преобразование речи в текст](https://cris.ai)
* [Преобразование текста в речь](text-to-speech.md)
* [Настраиваемое преобразование текста в речь](how-to-customize-voice-font.md)
* [Перевод речи](speech-translation.md) (не включает [перевод текста](../translator/translator-info-overview.md))

[Пакет SDK для службы "Речь"](speech-sdk.md) является функциональной заменой клиентских библиотек службы "Распознавание речи Bing", однако использует другой API.

## <a name="migration-strategies"></a>Стратегии миграции

Если вы или ваша организация приложений в среде разработки или рабочей среде, использующих API распознавания речи Bing, необходимо обновить их для использования служб речи, как можно скорее. См. в разделе [документации служб речи](index.yml) доступны пакеты SDK, примеры кода и учебные материалы.

Службы речи [интерфейсов REST API](rest-apis.md) совместимы с API распознавания речи Bing. Если вы сейчас используете REST API Bing для распознавания речи, необходимо только изменить конечную точку REST и переключитесь на ключ подписки службы распознавания речи.

Протоколы WebSockets службы распознавания речи, также совместимы с используемыми приложением распознавания речи Bing. Для разработки приложений мы рекомендуем использовать пакет SDK службы "Речь", а не протоколы WebSocket. Рекомендуется также перенести существующий код в этот пакет SDK. Однако, как и с REST API, существующий код, использующий службу "Распознавание речи Bing" через WebSocket, требует только изменения конечной точки и обновления ключа.

Если вы используете клиентскую библиотеку службы "Распознавание речи Bing" для определенного языка программирования, переход на [пакет SDK для службы "Речь"](speech-sdk.md) потребует внесения изменений в приложение из-за отличий в API. Пакет SDK службы "Речь" может упростить код, а также предоставить доступ к новым функциям.

В настоящее время пакет SDK службы "Речь" поддерживает C# (Windows 10, UWP, .NET Standard), Java (Android и пользовательские устройства), Objective C (iOS), C++ (Windows и Linux) и JavaScript. Интерфейсы API на всех платформах похожи, что упрощает разработку для нескольких платформ.

Службы речи не предлагают глобальной конечной точки. Необходимо определить, будет ли ваше приложение эффективно функционировать, используя одну региональную конечную точку для всего трафика. В противном случае используйте геолокацию, чтобы определить наиболее эффективную конечную точку. Требуется отдельная подписка службы распознавания речи в каждом регионе, которые можно использовать.

Если приложение использует длительные подключения и не поддерживает доступный пакет SDK, можно воспользоваться подключением WebSocket. Управляйте 10-минутным временем ожидания путем повторного подключения в нужное время.

Чтобы начать работу с пакетом SDK службы "Речь":

1. Загрузите [пакет SDK службы "Речь"](speech-sdk.md).
1. Рабочий процесс служб речи [краткие руководства](quickstart-csharp-dotnet-windows.md) и [учебники](how-to-recognize-intents-from-speech-csharp.md). Также ознакомьтесь с [примерами кода](samples.md), чтобы получить опыт работы с новыми интерфейсами API.
1. Обновите приложение, чтобы использовать службы распознавания речи.

Клиенты службы "Распознавание речи Bing" должны связаться со службой поддержки клиентов, воспользовавшись [запросом в службу поддержки](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/newsupportrequest). Кроме того, свяжитесь с нами, если для поддержки вам требуется [план технической поддержки](https://azure.microsoft.com/support/plans/).

Поддержка службы распознавания речи, пакет SDK и API, см. в статье службы речи [страница службы технической поддержки](support.md).

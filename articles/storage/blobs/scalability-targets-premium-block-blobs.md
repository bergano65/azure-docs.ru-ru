# <a name="scalability-targets-for-premium-block-blob-storage-accounts"></a>Целевые показатели масштабируемости для учетных записей хранения блочных BLOB-объектов класса Premium

Учетная запись хранения блочных BLOB-объектов уровня "Премиум" оптимизирована для приложений, использующих небольшие объекты в килобайтах. Это идеальный вариант для приложений, требующих высокой скорости транзакций или постоянного хранения с низкой задержкой. Хранилище BLOB-объектов производительности Premium предназначено для масштабирования с приложениями. Если в сценарии требуется развернуть приложения, требующие сотни тысяч запросов в секунду или петабайтов емкость хранилища, обратитесь в корпорацию Майкрософт, отправив запрос на поддержку в [портал Azure](https://portal.azure.com/?#blade/Microsoft_Azure_Support/HelpAndSupportBlade).

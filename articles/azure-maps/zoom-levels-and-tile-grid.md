# <a name="zoom-levels-and-tile-grid"></a>Уровни увеличения и параметры сетки
Карты Azure используется система координат проекции сферическую Меркатора (EPSG: 3857).

Карта мира разбита на квадратные плитки. Карты Azure предоставляет растровой и векторной плитки для 23 масштаба, от 0 до 22. На нулевом уровне масштаба вся карта мира помещается на одной плитке:

![Плитка карты мира](./media/zoom-levels-and-tile-grid/world0.png)

На первом уровне масштаба карта мира отображается в 2 фрагментах: квадрат 2 x 2.

![Плитка карты мира вверху слева](media/zoom-levels-and-tile-grid/world1a.png)     ![Плитка карты мира вверху справа](media/zoom-levels-and-tile-grid/world1c.png) 

![Плитка карты мира внизу слева](media/zoom-levels-and-tile-grid/world1b.png)     ![Плитка карты мира внизу справа](media/zoom-levels-and-tile-grid/world1d.png) 

Каждый уровень масштаба дополнительных quad разделяет плитки предыдущего разделяются, создавая сетку 2<sup>масштаб</sup> x 2<sup>масштаб</sup>. 22-й уровень масштаба — это сетка 2<sup>22</sup> x 2<sup>22</sup>, или 4 194 304 x 4 194 304 фрагмента (всего 17 592 186 044 416 фрагментов).

Элементы управления интерактивную карту "карты Azure" для Android поддержка через Интернет и масштаб уровни 25 масштаба, от 0 до 24. Несмотря на то, что пути данных доступна только на уровнях масштабирования в при наличии плитки.

Плитки формируются на основе уровня масштаба и координат x и y, которые соответствуют позиции плитки на сетке для этого уровня.

При выборе уровня масштаба для использования, помните, что каждое расположение — это Фиксированная позиция на его фрагмента мозаики. Это означает, что количество плиток, необходимое для отображения заданной территории, зависит от конкретного размещения сетки масштабирования на карте мира. Например, есть две точки, расстояние между которыми составляет 900 метров. Для отображения этого маршрута *может* понадобиться только три плитки на 17-ом уровне масштабирования. Но если западная точка находится а правой части своего фрагмента, а восточная — в левой части своего, может понадобиться четыре фрагмента:

![Демонстрация увеличения масштаба](media/zoom-levels-and-tile-grid/zoomdemo_scaled.png) 

После выбора уровня масштаба вычисляются значения x и y. Плитка верхнего левого в каждой сетке масштаба представляет x = 0, y = 0; Нижняя Плитка справа завершено на x = 2<sup>Масштаб -1</sup>, y = 2<sup>Масштаб-1</sup>.

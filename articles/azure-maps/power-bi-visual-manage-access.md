# <a name="manage-the-azure-maps-visual-within-your-organization"></a>Управление Azure Maps визуального элемента в Организации

Power BI предоставляет конструкторам и администраторам клиентов возможность управлять использованием Azure Mapsного визуального элемента. Ниже приводятся действия, которые может выполнять каждая роль.

### <a name="designer-options"></a>Возможности для авторов

В Power BI Desktop конструкторы могут отключить визуальный элемент Azure Maps на вкладке Безопасность. Выберите **File** &gt; **Параметры файла и параметры** , а затем выберите **Параметры** &gt; **Предварительная версия функций**. Если этот параметр отключен, Azure Maps не будет загружаться по умолчанию.  
  
![Панель параметров Power BI, показывающая параметры предварительной версии компонентов для Azure Maps визуального элемента](media/power-bi-visual/preview-options-panel.png)

## <a name="tenant-admin-options"></a>Параметры администратора клиента

В PowerBI.com Администраторы клиентов могут отключить визуальный элемент Azure Maps для всех пользователей. Выберите **Параметры** параметры &gt; **Admin** **Portal** &gt; **клиента**портала администрирования. Если этот параметр отключен, Power BI больше не будет отображать визуальный элемент Azure Maps в области визуализации.

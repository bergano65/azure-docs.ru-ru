# <a name="service-model-xml-schema-simple-types"></a>Простые типы в схеме XML модели службы

## <a name="fabricuri-simpletype"></a>FabricUri simpleType
Универсальный код ресурса (URI), который используется в качестве стабильного идентификатора служб в системе именования Microsoft Azure Service Fabric. 

### <a name="xml-source"></a>Источник XML
```xml
<xs:simpleType xmlns:xs="https://www.w3.org/2001/XMLSchema" xmlns="http://schemas.microsoft.com/2011/01/fabric" name="FabricUri">
    <xs:annotation>
      <xs:documentation>A URI that is used as a stable identifier of services by Microsoft Azure Service Fabric Naming system. </xs:documentation>
    </xs:annotation>
    <xs:restriction base="xs:anyURI"/>
  </xs:simpleType>
  
```

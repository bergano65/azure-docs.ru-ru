# <a name="sfctl-mesh"></a>sfctl mesh
Управление приложениями в службе "Сетка Service Fabric" и их удаление.

## <a name="subgroups"></a>Подгруппы
|Подгруппа|ОПИСАНИЕ|
| --- | --- |
| [app](service-fabric-sfctl-mesh-app.md) | Получение и удаление ресурсов приложения. |
| [code-package-log](service-fabric-sfctl-mesh-code-package-log.md) | Сведения о команде получения журналов для контейнера указанного пакета кода для данной реплики службы. |
| [deployment](service-fabric-sfctl-mesh-deployment.md) | Создание ресурсов службы "Сетка Service Fabric". |
| [шлюз](service-fabric-sfctl-mesh-gateway.md) | Получение и удаление ресурсов mesh gateway. |
| [network](service-fabric-sfctl-mesh-network.md) | Здесь приведены сведения о получении и удалении ресурсов mesh network. |
| [secret](service-fabric-sfctl-mesh-secret.md) | Получение и удаление ресурсов mesh secret. |
| [secretvalue](service-fabric-sfctl-mesh-secretvalue.md) | Получение и удаление ресурсов mesh secretvalue. |
| [service](service-fabric-sfctl-mesh-service.md) | Получение сведений о службе и списка служб ресурса приложения. |
| [service-replica](service-fabric-sfctl-mesh-service-replica.md) | Получение сведений о репликах и списка реплик данной службы в ресурсе приложения. |
| [volume](service-fabric-sfctl-mesh-volume.md) | Получение и удаление ресурсов тома. |

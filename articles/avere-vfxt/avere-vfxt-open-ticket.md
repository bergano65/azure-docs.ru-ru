# <a name="get-help-with-your-system"></a>Получение помощи по работе с системой

Если вам требуется помощь при работе с кластером Avere vFXT для Azure, вот несколько способов получить поддержку:

* **Проблема Avere vFXT** — используйте портал Azure, чтобы открыть запрос в службу поддержки для кластера Avere vFXT, как описано [ниже](#open-a-support-ticket-for-your-avere-vfxt).
* **Квота** — при наличии проблем, связанных с квотой, [запросите увеличение квоты](#request-a-quota-increase).
* **Документация и примеры** — если вы обнаружили проблемы с этой документацией или примерами, прокрутите страницу вниз и используйте раздел **Обратная связь** для поиска имеющихся проблем и при необходимости сообщите о новой проблеме.  

## <a name="open-a-support-ticket-for-your-avere-vfxt"></a>Открытие запроса в службу поддержки для Avere vFXT

Если возникают проблемы при развертывании или использовании Avere vFXT, отправьте запрос на поддержку с помощью портала Azure.  

Выполните следующие действия, чтобы в запросе в службу поддержки был указан нужный ресурс кластера. Прикрепление ресурса к запросу помогает нам перенаправить его подходящему сотруднику службы поддержки. 

1. На странице [https://portal.azure.com](https://portal.azure.com) выберите **Группы ресурсов**.

   ![снимок экрана меню в левой области портала Azure с выделенным разделом "Группы ресурсов"](media/avere-vfxt-ticket-rg.png)

1. Перейдите в группу ресурсов, которая содержит кластер vFXT, где возникла проблема, и выберите одну из виртуальных машин Avere.

    ![снимок экрана области обзора группы ресурсов на портале Azure с выделенной виртуальной машиной](media/avere-vfxt-ticket-vm.png)

1. На странице виртуальной машины прокрутите до нижней части левой области, а затем выберите **Новый запрос на получение поддержки**.

    ![Снимок экрана портала Azure: страница виртуальной машины (той же, которая была показана на предыдущем снимке экрана). Нижняя область меню слева с выделенным параметром "Новый запрос на получение поддержки".](media/avere-vfxt-ticket-request.png)

1. На первой странице запроса на поддержку выберите параметр **Все службы**, а затем в списке **хранилищ** выберите **Avere vFXT**.

    ![снимок экрана нового запроса на поддержку на портале Azure с заголовком "Основные сведения" и выделенным элементом "Служба". Выбран параметр "Все службы", а в поле раскрывающегося меню выбрано значение Avere vFXT](media/avere-vfxt-ticket-service.png)

1. На второй странице выберите наиболее подходящие тип и категорию проблемы. Добавьте краткое название и описание, которое включает время возникновения проблемы. 

   ![снимок экрана страницы нового запроса на поддержку с заголовком "Проблема", содержащей несколько полей, которые нужно заполнить](media/avere-vfxt-ticket-problem.png)

1. На третьей странице введите свои контактные данные и нажмите кнопку **Создать**. На ваш адрес электронной почты будут отправлены подтверждение и номер запроса, после чего с вами свяжется сотрудник службы поддержки.

## <a name="request-a-quota-increase"></a>Запрос на увеличение квоты

Ознакомьтесь с разделом о [квоте для кластера vFXT](avere-vfxt-prereqs.md#quota-for-the-vfxt-cluster), чтобы узнать, какие компоненты требуются для развертывания кластера Avere vFXT для Azure. Вы можете [запросить увеличение квоты](https://docs.microsoft.com/azure/azure-supportability/resource-manager-core-quotas-request) на портале Azure.

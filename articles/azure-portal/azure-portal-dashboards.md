# <a name="create-and-share-dashboards-in-the-azure-portal"></a>Создание панелей мониторинга на портале Azure и предоставление общего доступа к ним

Панели мониторинга — это организованное и упорядоченное представление облачных ресурсов в портал Azure. Используйте панели мониторинга в качестве рабочей области, где можно быстро запускать задачи для повседневных операций и мониторинга ресурсов. Например, создавайте настраиваемые панели мониторинга на основе проектов, задач или ролей пользователей.

Портал Azure предоставляет панель мониторинга по умолчанию в качестве отправной точки. Панель мониторинга по умолчанию можно изменить. Создавайте и настраивайте дополнительные панели мониторинга, публикуйте панели мониторинга и делитесь ими, чтобы сделать их доступными для других пользователей. В этой статье описывается создание новой панели мониторинга, Настройка интерфейса, а также публикация и совместное использование панелей мониторинга.

## <a name="create-a-new-dashboard"></a>Создание панели мониторинга

В этом примере мы создадим новую, закрытую панель мониторинга и присваиваем имя. Чтобы приступить к работе, выполните эти действия:

1. Войдите на [портал Azure](https://portal.azure.com).

1. В меню портал Azure выберите **панель мониторинга**. Возможно, для представления по умолчанию уже задана панель мониторинга.

    ![Открытие панели мониторинга](./media/azure-portal-dashboards/portal-menu-dashboard.png)

1. Выберите **создать панель мониторинга**.

    ![Снимок экрана новой панели мониторинга](./media/azure-portal-dashboards/create-new-dashboard.png)

    Это действие открывает **коллекцию плиток**, из которой вы выберете плитки и пустую сетку, в которой вы размещаете плитки.

    ![Снимок экрана: Коллекция плиток и пустая сетка](./media/azure-portal-dashboards/dashboard-name.png)

1. Выберите текст **Моя панель мониторинга** в метке панели мониторинга и введите имя, которое поможет легко определить настраиваемую панель мониторинга.

1. Выберите **Готово Настройка** в заголовке страницы, чтобы выйти из режима редактирования.

Теперь в представлении "панель мониторинга" отображается новая панель мониторинга. Щелкните стрелку рядом с именем панели мониторинга, чтобы просмотреть доступные панели мониторинга. Список может включать панели мониторинга, которые были созданы и предоставлены другими пользователями.

## <a name="edit-a-dashboard"></a>Изменение панели мониторинга

Теперь изменим панель мониторинга, чтобы добавлять, изменять размеры и упорядочивать плитки, представляющие ресурсы Azure.

### <a name="add-tiles-from-the-dashboard"></a>Добавление плиток с панели мониторинга

Чтобы добавить плитки на панель мониторинга, выполните следующие действия.

1. Выберите ![ изменить значок ](./media/azure-portal-dashboards/dashboard-edit-icon.png) **изменить** в заголовке страницы.

    ![Снимок экрана: изменение выделения панели мониторинга](./media/azure-portal-dashboards/dashboard-edit.png)

1. Просмотрите **коллекцию плиток** или используйте поле поиска, чтобы найти нужную плитку.

1. Выберите **Добавить** , чтобы добавить плитку на панель мониторинга с размером и расположением по умолчанию. Или перетащите плитку в сетку и поместите ее там, где нужно.

> [!TIP]
> Если вы работаете с несколькими организациями, добавьте плитку " **удостоверение Организации** " на панель мониторинга, чтобы четко продемонстрировать, к какой организации принадлежат ресурсы.

### <a name="add-tiles-from-a-resource-page"></a>Добавление плиток со страницы ресурсов

Существует альтернативный способ добавления плиток на панель мониторинга. Многие страницы ресурсов содержат значок канцелярской кнопки на панели команд. Если выбрать значок, плитка, представляющая исходную страницу, будет закреплена на панели мониторинга, которая в данный момент активна. 

![Снимок экрана: панель команд страницы со значком закрепления](./media/azure-portal-dashboards/dashboard-pin-blade.png)

### <a name="resize-or-rearrange-tiles"></a>Изменение размера или переупорядочивания плиток

Чтобы изменить размер плитки или Переупорядочить плитки на панели мониторинга, выполните следующие действия.

1. Выберите ![ изменить значок ](./media/azure-portal-dashboards/dashboard-edit-icon.png) **изменить** в заголовке страницы.

1. Выберите контекстное меню в верхнем правом углу плитки. Затем выберите размер плитки. Плитки, которые поддерживают любой размер, также содержат "Handle" в правом нижнем углу, который позволяет перетаскивать плитку до нужного размера.

    ![Снимок экрана панели мониторинга с открытым меню "Размер плитки"](./media/azure-portal-dashboards/dashboard-tile-resize.png)

1. Выберите плитку и перетащите ее в новое место в сетке, чтобы упорядочить панель мониторинга.

### <a name="additional-tile-configuration"></a>Дополнительная конфигурация плитки

Для отображения нужных сведений некоторым плиткам может потребоваться дополнительная настройка. Например, плитку **диаграммы метрик** необходимо настроить для отображения метрики из **Azure Monitor**. Можно также настроить данные плитки, чтобы переопределить параметры времени по умолчанию для панели мониторинга.

Все плитки, которые необходимо настроить, отображают заголовок **плитки настройки** до тех пор, пока вы не настроите плитку. Чтобы настроить плитку, сделайте следующее:

1. Выберите **Готово Настройка** в заголовке страницы, чтобы выйти из режима редактирования.

1. Выберите баннер, а затем выполните необходимую настройку.

    ![Снимок экрана с плиткой, требующей настройки](./media/azure-portal-dashboards/dashboard-configure-tile.png)

> [!NOTE]
> Плитка Markdown позволяет отображать настраиваемое статическое содержимое на панели мониторинга. Это могут быть основные инструкции, изображение, набор гиперссылок или даже Контактная информация. Дополнительные сведения об использовании плитки Markdown см. в статье [Использование плитки Markdown на панелях мониторинга Azure для отображения пользовательского содержимого](azure-portal-markdown-tile.md).

### <a name="customize-tile-data"></a>Настройка данных плитки

Данные на панели мониторинга автоматически показывают активность за последние 24 часа. Чтобы отобразить другой интервал времени только для этой плитки, выполните следующие действия.

1. Выберите пункт **настроить данные плитки** в контекстном меню или ![ Фильтр фильтра ](./media/azure-portal-dashboards/dashboard-filter.png) в левом верхнем углу плитки.

    ![Снимок экрана: контекстное меню плитки](./media/azure-portal-dashboards/dashboard-customize-tile-data.png)

1. Установите флажок, чтобы **переопределить параметры времени панели мониторинга на уровне плитки**.

    ![Снимок экрана диалогового окна для настройки параметров времени плитки](./media/azure-portal-dashboards/dashboard-override-time-settings.png)

1. Выберите интервал времени, который будет отображаться для этой плитки. Вы можете выбрать последние 30 минут в течение последнего 30 дней или определить пользовательский диапазон.

1. Выберите гранулярность времени для вывода. Можно отобразить любое место от одного минуты к одному месяцу.

1. Нажмите кнопку **Применить**.

## <a name="delete-a-tile"></a>Удаление плитки

Чтобы удалить плитку из панели мониторинга, выполните следующие действия.

* Выберите контекстное меню в правом верхнем углу плитки, а затем выберите **удалить с панели мониторинга**. или

* Выберите ![ изменить значок ](./media/azure-portal-dashboards/dashboard-edit-icon.png) **изменить** , чтобы перейти в режим настройки. Наведите указатель мыши в правом верхнем углу плитки, а затем щелкните значок Удалить значок удаления, ![ ](./media/azure-portal-dashboards/dashboard-delete-icon.png) чтобы удалить плитку из панели мониторинга.

   ![Снимок экрана, показывающий, как удалить плитку из панели мониторинга](./media/azure-portal-dashboards/dashboard-delete-tile.png)

## <a name="clone-a-dashboard"></a>Клонирование панели мониторинга

Чтобы использовать существующую панель мониторинга в качестве шаблона для новой панели мониторинга, выполните следующие действия.

1. Убедитесь, что в представлении "панель мониторинга" отображается панель мониторинга, которую требуется скопировать.

1. В заголовке страницы выберите клонировать значок клона ![ ](./media/azure-portal-dashboards/dashboard-clone.png) **Clone**.

1. Копия панели мониторинга, называемая **клоном** *имени панели мониторинга* , открывается в режиме редактирования. Выполните действия, описанные в этой статье, чтобы переименовать и настроить панель мониторинга.

## <a name="publish-and-share-a-dashboard"></a>Публикация и совместное использование панели мониторинга

При создании панели мониторинга она является закрытой по умолчанию, что означает, что это единственный пользователь, который может ее видеть. Чтобы сделать панели мониторинга доступными для других пользователей, их можно опубликовать и предоставить к ним общий доступ. Дополнительные сведения см. в статье [предоставление общего доступа к панелям мониторинга Azure с помощью управления доступом на основе ролей](azure-portal-dashboard-share-access.md).

### <a name="open-a-shared-dashboard"></a>Открытие общей панели мониторинга

Чтобы найти и открыть общую информационную панель, выполните следующие действия.

1. Щелкните стрелку рядом с именем панели мониторинга.

1. Выберите из отображаемого списка панелей мониторинга. Если панель мониторинга, которую требуется открыть, не указана:

    1. Выберите **Обзор все панели мониторинга**.

        ![Снимок экрана: меню выбора панели мониторинга](./media/azure-portal-dashboards/dashboard-browse.png)

    1. В поле **тип** выберите **Общие панели мониторинга**.

        ![Снимок экрана: меню выбора всех панелей мониторинга](./media/azure-portal-dashboards/dashboard-browse-all.png)

    1. Выберите одну или несколько подписок. Можно также ввести текст для фильтрации панелей мониторинга по имени.

    1. Выберите панель мониторинга из списка общих панелей мониторинга.

## <a name="delete-a-dashboard"></a>Удаление панели мониторинга

Чтобы окончательно удалить частную или общую панель мониторинга, выполните следующие действия.

1. Выберите панель мониторинга, которую требуется удалить, из списка рядом с именем панели мониторинга.

1. Выберите ![ Удалить значок ](./media/azure-portal-dashboards/dashboard-delete-icon.png) **Удалить** из заголовка страницы.

1. Для закрытой панели мониторинга нажмите кнопку **ОК** в диалоговом окне подтверждения, чтобы удалить панель мониторинга. Для общей панели мониторинга в диалоговом окне подтверждения установите флажок, чтобы убедиться, что опубликованная панель мониторинга больше не будет доступна для просмотра другими пользователями. Нажмите кнопку **ОК**.

    ![Снимок экрана с подтверждением удаления](./media/azure-portal-dashboards/dashboard-delete-dash.png)

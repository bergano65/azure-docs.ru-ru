# <a name="configure-usage-settings-and-policies"></a>Настройка параметров и политик использования
В этой статье описывается, как добавить и зарегистрировать пользователей в лаборатории, контролировать количество часов для использования виртуальной машины и многое другое. 


## <a name="add-users-to-the-lab"></a>Добавление пользователей в лабораторию
Если у вас включен параметр **Restrict access** (Ограничить доступ), добавьте в список пользователей (адреса электронной почты).

1. В меню слева выберите **Пользователи**.
2. На панели инструментов выберите **Добавить пользователей**. 

    ![Кнопка "Добавить пользователей"](../media/how-to-configure-student-usage/add-users-button.png)
1. На странице **Добавление пользователей** введите адреса электронной почты пользователей в отдельных строках или одной строке, разделяя их точкой с запятой. 

    ![Добавление адресов электронной почты пользователей](../media/how-to-configure-student-usage/add-users-email-addresses.png)
4. Щелкните **Сохранить**. В списке отобразятся адреса электронной почты и состояние пользователей (зарегистрирован или нет). 

    ![Список пользователей](../media/how-to-configure-student-usage/users-list-new.png)

## <a name="send-registration-link-to-students"></a>Отправка ссылки для регистрации учащимся
В следующей процедуре содержатся шаги для отправки ссылки регистрации пользователям. Если **Ограничение доступа** включено для лаборатории, только пользователи из списка пользователей могут использовать ссылку регистрации для регистрации в лаборатории. 

1. Перейдите к представлению **Пользователи**, выбрав **Пользователи** в меню слева. 
2. Щелкните плитку **Получить ссылку для регистрации**.

    ![Ссылка для регистрации учащихся](../media/tutorial-setup-classroom-lab/dashboard-user-registration-link.png)
1. В диалоговом окне **User registration** (Регистрация пользователей) щелкните **Копировать**. Ссылка скопируется в буфер обмена. Вставьте ее в редактор электронной почты и отправьте электронное сообщение учащемуся. 

    ![Ссылка для регистрации учащихся](../media/tutorial-setup-classroom-lab/registration-link.png)
2. В диалоговом окне **Регистрация пользователей** нажмите кнопку **Закрыть**. 
4. Отправьте учащемуся ссылку для регистрации, чтобы он мог зарегистрироваться для прохождения курса. Если включен параметр **Ограничить доступ** и в списке есть пользователи, выполните следующие действия:
    1. Щелкните **адрес электронной почты** пользователя в списке. 
    2. Запустится программа электронной почты по умолчанию и отобразится окно сообщения с заполненной строкой **Кому**. 
    3. Вставьте в сообщение **URL-адрес для регистрации**, скопированный ранее. 
    4. Отправьте **сообщение электронной почты**. 

## <a name="view-users-registered-with-the-lab"></a>Просмотр пользователей, зарегистрированных в лаборатории

Выберите в меню слева раздел **Пользователи**, чтобы просмотреть список зарегистрированных в лаборатории пользователей. 

![Список пользователей, зарегистрированных в лаборатории](../media/how-to-configure-student-usage/users-list-new.png)

## <a name="set-quotas-per-user"></a>Настройка квот для пользователей
Можно установить квоты на пользователя, выполнив следующие действия: 

1. В меню слева выберите **Пользователи**.
2. Выберите **Quota per user: unlimited** (Квота на пользователя: нограничено) на панели инструментов. 
3. На странице **Quota per user** (Квота на пользователя) выберите один из следующих параметров. 
    1. **None**. Пользователи могут использовать свои виртуальные машины только в запланированное время или когда владелец лаборатории включает для них виртуальные машины.
    2. **Unlimited (default)** (Без ограничений (по умолчанию)). Пользователи могут использовать виртуальные машины без ограничения по времени.
    3. **Specify number of hours per user** (Задать количество часов на одного пользователя). Пользователи могут использовать свои виртуальные машины в течение заданного времени (указано ниже) в дополнение к расписанию. При выборе этого параметра введите **количество часов** в текстовом поле. 

        ![Количество часов на одного пользователя](../media/how-to-configure-student-usage/number-of-hours-per-user.png)
    4. Щелкните **Сохранить**. 
5. Измененные значения теперь отображаются на панели инструментов. **Quota per user: &lt;number of hours&gt;**. (Квота на пользователя: <количество часов>) 

    ![Квота для пользователя](../media/how-to-configure-student-usage/quota-per-user.png)

> [!IMPORTANT]
> Запланированное время запуска виртуальных машин не учитывается в [квоте, выделенной пользователю](how-to-create-schedules.md). Квота рассчитывается на время, которое выходит за пределы расписания и которое учащийся тратит на виртуальные машины. 

### <a name="add-users-by-uploading-a-csv-file"></a>Добавление пользователей путем отправки CSV-файла
Можно также добавить пользователей, передав CSV-файл с адресами электронной почты пользователей.

1. Создайте CSV-файл с помощью адреса электронной почты пользователей в одном столбце.

    ![Квота для пользователя](../media/how-to-configure-student-usage/csv-file-with-users.png)
2. На странице лаборатории **Пользователи** на панели инструментов выберите **Отправить CSV**.

    ![Кнопка "Отправить CSV"](../media/how-to-configure-student-usage/upload-csv-button.png)
3. Выберите CSV-файл с адресами электронной почты пользователя. Щелкнув **Открыть** после того, как вы выбрали CSV-файл, вы увидите следующее окно **Добавление пользователей**. Список адресов электронной почты будет заполнен из CSV-файла. 

    ![Окно "Добавление пользователей", заполненное адресами электронной почты из CSV-файла](../media/how-to-configure-student-usage/add-users-window.png)
4. В окне **Добавление пользователей** выберите **Сохранить**. 
5. Убедитесь, что пользователи отображаются в списке пользователей. 

    ![Список добавленных пользователей](../media/how-to-configure-student-usage/list-of-added-users.png)

## <a name="manage-user-vms"></a>Управление пользовательскими виртуальными машинами
После регистрации учащихся в Службах лабораторий Azure с помощью предоставленной вами ссылки для регистрации вы сможете просмотреть виртуальные машины, назначенные учащимся, на вкладке **Виртуальные машины**. 

![Виртуальные машины, назначенные учащимся](../media/how-to-manage-classroom-labs/virtual-machines-students.png)

Вы можете выполнить следующие действия с виртуальной машиной учащегося: 

- Остановить виртуальную машину, если она запущена. 
- Запустить виртуальную машину, если она остановлена. 
- Подключитесь к виртуальной машине. 
- Удалите виртуальную машину. 
- проверить количество часов, в течение которых пользователи использовали виртуальную машину. 

## <a name="update-number-of-virtual-machines-in-lab"></a>Обновление количества виртуальных машин в лаборатории
Чтобы обновить количество виртуальных машин в лаборатории, на странице **Виртуальные машины** сделайте следующее.

1. В меню слева выберите **Виртуальные машины**. 
2. На панели инструментов выберите **Lab capacity: &lt;number&gt; machine(s)** (Емкость лаборатории: <количество> компьютеров). 
3. Введите **количество** виртуальных машин.
4. Щелкните **Сохранить**.

    ![Виртуальные машины в лаборатории](../media/how-to-configure-student-usage/number-virtual-machines.png)

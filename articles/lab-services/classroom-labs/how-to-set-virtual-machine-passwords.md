# <a name="set-up-and-manage-virtual-machine-pool"></a>Настройка пула виртуальных машин и управление им 
В этой статье показано, как выполнить следующие задачи.

- Увеличение числа виртуальных машин (ВМ) в лаборатории
- Запуск всех виртуальных машин или выбранных виртуальных машин 
- Сброс виртуальных машин

## <a name="update-the-lab-capacity"></a>Обновление емкости лаборатории
Чтобы увеличить или уменьшить емкость лаборатории (число виртуальных машин в лаборатории), выполните следующие действия.

1. На странице **пул виртуальных машин** выберите **емкость лаборатории: &lt;число&gt; машин**.
2. Введите новое **число виртуальных машин** в лаборатории. Это число должно быть больше или равно числу пользователей, зарегистрированных в лаборатории. 
3. Затем нажмите кнопку **Сохранить**. 

    ![Кнопка "запустить все"](../media/how-to-set-virtual-machine-passwords/number-of-vms-in-lab.png)
4. Если вы увеличили емкость, вы видите виртуальную машину или виртуальные машины, которые будут созданы. Если вы не видите новую виртуальную машину в списке, обновите страницу. 

    ![Создаваемая виртуальная машина](../media/how-to-set-virtual-machine-passwords/vm-being-created.png)

## <a name="start-vms"></a>Запуск виртуальных машин

### <a name="start-ot-stop-all-vms"></a>Запуск на всех виртуальных машинах
1. Перейдите на страницу **пула виртуальных машин** . 
2. Выберите **запустить все** на панели инструментов. 

    ![Кнопка "запустить все"](../media/how-to-set-virtual-machine-passwords/start-all-vms-button.png)
3. После запуска всех виртуальных машин можно отключить все виртуальные машины, нажав кнопку « **все** » на панели инструментов. 

    ![Кнопка "закрыть все"](../media/how-to-set-virtual-machine-passwords/stop-all-vms-button.png)

### <a name="start-selected-vms"></a>Запуск выбранных виртуальных машин
Существует два способа запуска выбранных виртуальных машин (один или несколько). Первый способ — выбрать виртуальную машину или виртуальные машины в списке, а затем нажать кнопку **запустить** на панели инструментов. 

Второй способ — выбрать одну или несколько виртуальных машин в списке и переключить кнопку в столбце **состояние** . 

![Запуск выбранных виртуальных машин](../media/how-to-set-virtual-machine-passwords/start-selected-vms.png)

Аналогичным образом можно отключить одну или несколько виртуальных машин, переключив кнопку в столбце **состояние** или выбрав пункт **Закрыть** на панели инструментов. 

## <a name="reset-vms"></a>Сброс виртуальных машин
Чтобы сбросить одну или несколько виртуальных машин, выберите их в списке и нажмите кнопку **Сброс** на панели инструментов. 

![Сброс выбранных виртуальных машин](../media/how-to-set-virtual-machine-passwords/reset-vm-button.png)

В диалоговом окне **Сброс виртуальных машин** выберите **сбросить**. 

![Диалоговое окно "сброс виртуальной машины"](../media/how-to-set-virtual-machine-passwords/reset-vms-dialog.png)



## <a name="set-password-for-vms"></a>Установка пароля для виртуальных машин
Владелец лаборатории (преподаватель) может установить или сбросить пароль для виртуальных машин во время создания лаборатории (мастер создания лабораторий) или после создания лаборатории на странице **шаблона** . 

### <a name="set-password-at-the-time-of-lab-creation"></a>Задать пароль во время создания лаборатории
Владелец лаборатории (преподаватель) может задать пароль для виртуальных машин в лаборатории на странице **учетные данные виртуальной машины** мастера создания лабораторий.

![Новое окно лаборатории](../media/tutorial-setup-classroom-lab/virtual-machine-credentials.png)

Включив или отключив параметр **использовать один и тот же пароль для всех виртуальных машин** на этой странице, преподаватель может выбрать один и тот же пароль для всех виртуальных машин в лаборатории или разрешить учащимся устанавливать пароли для их виртуальных машин. По умолчанию этот параметр включен для всех образов операционных систем Windows и Linux, кроме Ubuntu. Если этот параметр отключен, студентам будет предложено задать пароль при первой попытке подключения к виртуальной машине. 

### <a name="reset-password-later"></a>Сбросить пароль позже

1. На странице **шаблона** лаборатории на панели инструментов выберите **Сброс пароля** . 
1. В диалоговом окне " **Сброс пароля** " введите пароль и щелкните " **сбросить пароль**".
    
    ![Диалоговое окно «Установка пароля»](../media/how-to-set-virtual-machine-passwords/set-password.png)

## <a name="connect-to-student-vms"></a>Подключение к виртуальным машинам учащихся
Создатель лаборатории (Instructor/профессор) может подключиться к виртуальной машине учащихся, если выполняются следующие условия. 

- При создании лаборатории был выбран параметр **использовать тот же пароль для всех виртуальных машин**
- Виртуальная машина запущена 

 Чтобы подключиться к виртуальной машине учащегося, наведите указатель мыши на виртуальную машину в списке и нажмите кнопку компьютер.  

![Кнопка "подключиться к виртуальной машине учащегося"](../media/how-to-set-virtual-machine-passwords/connect-student-vm.png)

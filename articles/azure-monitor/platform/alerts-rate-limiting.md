# <a name="rate-limiting-for-voice-sms-emails-azure-app-push-notifications-and-webhook-posts"></a>Ограничение частоты отправки для голосовых сообщений, SMS-сообщений, сообщений электронной почты, push-уведомлений приложений Azure и записей веб-перехватчиков
Ограничение частоты означает приостановку уведомлений, которые слишком часто отправляются на указанный номер телефона, адрес электронной почты или устройство. Ограничение скорости гарантирует управляемость и действенность оповещений.

Пороговые значения частоты:

 - **SMS:** не более 1 SMS каждые 5 минут.
 - **Голосовые сообщения:** не более 1 голосового звонка каждые 5 минут.
 - **Электронная почта:** не более 100 сообщений электронной почты в течение часа.
 
 Для других действий ограничений нет.

## <a name="rate-limit-rules"></a>Правила ограничения частоты
- Нужный номер телефона или адрес электронной почты ограничивается, если частота превысит пороговое значение.
- Номер телефона или адрес электронной почты может быть частью групп действий в нескольких подписках. Ограничение частоты применяется ко всем подпискам по достижении порогового значения, даже если сообщения отправляются из нескольких подписок.
- Если для адреса электронной почты ограничена частота, отправляется соответствующее уведомление. Состояния сообщений, когда истекает срок действия ограничения частоты.

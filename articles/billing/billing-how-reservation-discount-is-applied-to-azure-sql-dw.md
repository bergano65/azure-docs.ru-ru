# <a name="how-reservation-discounts-apply-to-azure-sql-data-warehouse"></a>Применение скидки резервирования для хранилища данных SQL Azure

После покупки емкости хранилища данных SQL защищены скидка на резервирование автоматически применяется к хранилищам данных, которые существуют в этом регионе. Скидка на резервирование применяется к потреблению ресурсов по счетчику cDWU хранилище данных SQL. Хранилища и сети, оплачивается по ставкам повременной оплаты.

## <a name="reservation-discount-application"></a>Правила применения скидок на резервирование

Скидка зарезервированная емкость хранилища данных SQL применяется к работающим складов на почасовой основе. Если у вас нет хранилища, развернутых на один час, зарезервированный объем ресурсов впустую за этот час. Он не применяется.

После приобретения резервирования, который вы приобретаете соответствует потреблению ресурсов хранилища данных SQL, выполнив хранилищ в любой момент времени. Если вы завершаете работу некоторых хранилищ, затем скидки резервирования автоматически применяется к других сопоставления хранилищ.

Для хранилищ, которые не выполняются за полный час резервирование автоматически применяется к другим экземплярам сопоставления в течение этого часа.

## <a name="discount-examples"></a>Скидки примеры

Ниже приведены примеры, как применяется скидка зарезервированная емкость хранилища данных SQL, в зависимости от развертывания.

- **Пример 1**: Можно приобрести 5 единиц емкости 100 cDWU зарезервировано. Запустить экземпляр хранилища данных SQL DW1500c на один час. В этом случае использования создается на 15 единиц 100 использования cDWU. Скидка на резервирование применяется к 5 единиц, которые вы использовали. Оплата определяется с помощью ставкам повременной оплаты для оставшихся 10 единиц, 100 cDWU использования, который использовался.

- **Пример 2**. Можно приобрести 5 единиц емкости 100 cDWU зарезервировано. Вы запустите два экземпляра хранилища данных SQL DW100c на один час. В этом случае за 1 единицу 100 использования cDWU создаются два события использования. Оба события использования получите скидки на зарезервированный объем ресурсов. Оставшиеся 3 единицы емкости 100 зарезервированных cDWU будут потрачены впустую и не переносятся для использования в будущем.

- **Пример 3**: Вы приобретаете 1 единица мощности 100 зарезервированных cDWU. Вы запустите два экземпляра хранилища данных SQL DW100c. Каждый выполняется в течение 30 минут. В этом случае оба события использования получите скидки на зарезервированный объем ресурсов. Нет плата за использование начисляется по ставкам оплаты по мере использования.

# <a name="azure-software-plan-discount"></a>Скидка план программного обеспечения в Azure

Планы программного обеспечения в Azure для SUSE и RedHat, резервирования, которые применяются для развернутых виртуальных машин. Скидка план программного обеспечения применяется к использованию программного обеспечения развернутых виртуальных машин, которые соответствуют резервирование.

При завершении работы виртуальной Машины, скидка автоматически применяется к другой виртуальной Машине, сопоставления, если он доступен. План программного обеспечения входит стоимость программным обеспечением на виртуальной Машине. Другие расходы, такие как вычисления, хранения и сети, взимается отдельно.

Чтобы купить подходящий план, необходимо понимать использование виртуальных Машин и число виртуальных ЦП на этих виртуальных машинах. Следуйте приведенным ниже, чтобы определить, что планируют приобрести, основанные на данных об использовании.

## <a name="how-reservation-discount-is-applied"></a>Применение скидки резервирования

Скидка резервирования "*использования ИТ или потерять ИТ-*«. Таким образом Если у вас нет сопоставления ресурсов для данного часа, тогда у вас не количество резервирований для этого часа. Не удается выполнить пересылать неиспользуемые зарезервированные часов.

При завершении работы ресурса скидка на резервирование автоматически применяется к другим соответствующий ресурс в указанной области. Если нет соответствия ресурсов находятся в указанной области, а затем зарезервированный часы: *потеряны*.

## <a name="review-redhat-vm-usage-before-you-buy"></a>Просмотрите сведения об использовании виртуальной Машины RedHat в том случае, перед покупкой

Получить данные об использовании имя продукта и приобрести план RedHat с один и тот же тип и размер.

Например, если потребление имеет продукта **Red Hat Enterprise Linux — 1 – 4 виртуальных ЦП виртуальной Машины лицензии**, следует приобрести **Red Hat Enterprise Linux** для **1 – 4 виртуальных ЦП виртуальной Машины**.

## <a name="review-suse-vm-usage-before-you-buy"></a>Просмотрите использование виртуальных Машин SUSE перед покупкой

Получить данные об использовании имя продукта и приобрести план SUSE с один и тот же тип и размер.

Например, если используется для продукта **SUSE Linux Enterprise Server приоритет — виртуальный ЦП 2 – 4 поддержки виртуальных Машин**, следует приобрести **SUSE Linux Enterprise Server приоритет** для **2 – 4 виртуальных ЦП**.

## <a name="discount-applies-to-different-vm-sizes-for-suse-plans"></a>Скидка применяется к различных размеров виртуальных Машин для планов SUSE

Как и зарезервированные экземпляры виртуальных машин, план SUSE предлагает гибкость размера экземпляра. Это означает, что ваша скидка применяется даже в случае развертывания виртуальной машины с разным количеством виртуальных процессоров. Скидка применяется к виртуальным машинам разных размеров в рамках плана программного обеспечения.

Сумма скидки зависит от коэффициента, указанного в следующих таблицах. Коэффициент сравнивает относительный охват для каждой метрики в этой группе. Коэффициент зависит от виртуальных ЦП виртуальной машины. Используйте значение коэффициента для вычисления того, на сколько экземпляров виртуальных машин можно получить скидку по плану SUSE Linux.

Например, если вы покупаете план для SUSE Linux Enterprise Server для HPC (цен. категория "Приоритетный") для виртуальной машины с 3 или 4 виртуальными ЦП, то коэффициент для этого резервирования — 2. Скидка покрывает стоимость программного обеспечения SUSE для:

- 2 развернутых виртуальных машин с 1 или 2 виртуальными ЦП;
- 1 развернутой виртуальной машины с 3 или 4 виртуальными ЦП;
- примерно 77 % (0,77) виртуальной машины с 5 или более виртуальными ЦП.

Коэффициент для 5 или более виртуальных ЦП составляет 2,6. Так что резервирование для SUSE с виртуальной машиной с 5 или более виртуальными ЦП покрывает только часть затрат на программное обеспечение и составляет около 77 %.

# <a name="change-the-profile-information-for-your-azure-account"></a>Изменение сведений в профиле для учетной записи Azure

Корпорация Майкрософт использует адрес электронной почты в профиле администратора учетной записи для отправки важных уведомлений о выставлении счетов, службах и рекомендациях, касающихся учетной записи Azure. Мы рекомендуем указывать адрес электронной почты, который будет регулярно проверяться. Как администратор учетной записи, вы также можете обновить свой профиль, чтобы изменить почтовый адрес и номер телефона для учетной записи.

Корпорация Майкрософт использует адрес электронной почты в профиле пользователя, который не является администратором учетной записи, для отправки уведомлений и рекомендаций, связанных с учетной записью. Все уведомления о выставлении счетов отправляются администратору учетной записи. Вы также можете обновить свой профиль, чтобы изменить свой почтовый адрес и номер телефона, но это не изменит почтовый адрес или номер телефона учетной записи.

## <a name="change-your-profile-information"></a>Изменение данных в профиле

1. Войдите в [Центр учетных записей Azure](https://account.azure.com/subscriptions).
1. Нажмите кнопку **Изменить сведения**, а затем обновите сведения в **профиле**.

   ![Пример данных профиля Azure](./media/billing-how-to-change-azure-account-profile/profile.png)

### <a name="send-notifications-to-an-additional-email-address"></a>Отправка уведомлений на дополнительный адрес электронной почты

Вы можете получать уведомления службы и рекомендации для учетной записи, отправленные на адрес электронной почты в дополнение к адресу электронной почты администратора учетной записи, который связан с вашей учетной записью. На адрес электронной почты администратора учетной записи поступают важные уведомления обо всех подписках в учетной записи, включая обновления, связанные с выставлением счетов.

### <a name="change-the-account-administrators-email-address-in-your-profile"></a>Изменение адреса электронной почты администратора учетной записи в профиле

Администратор учетной записи — это человек, который настраивает учетную запись Azure и получает электронные уведомлений обо всех подписках в учетной записи. Рекомендуется указать контактный адрес электронной почты для регулярной проверки администратором учетной записи.

### <a name="update-your-sign-in-email-address"></a>Обновление адреса электронной почты для входа

При обновлении адреса электронной почты в профиле адрес электронной почты для входа не обновляется. Чтобы изменить адрес электронной почты для входа, необходимо передать право владения учетной записью.

### <a name="update-the-billing-address-for-your-credit-card"></a>Обновление адреса выставления счетов для кредитной карты

Сведения об обновлении данных о счетах см. в статье [Изменение кредитной карты, используемой для оплаты подписки Azure](billing-how-to-change-credit-card.md).

### <a name="update-your-countryregion"></a>Обновление страны/региона

В связи с техническими ограничениями нельзя изменить страну или регион имеющейся учетной записи. Однако вы можете создать учетную запись в нужной стране или регионе, а затем обратиться в службу поддержки Azure с просьбой о переносе служб в эту учетную запись.

## <a name="service-and-marketing-emails"></a>Служба и маркетинговые сообщения электронной почты

На [портале Azure](https://portal.azure.com) вам предлагается проверять или обновлять адрес электронной каждые 90 дней. Корпорация Майкрософт отправляет электронные письма на адрес электронной почты профиля с информацией, связанной с учетной записью Azure, для:

- Уведомления службы
- Оповещения безопасности
- выставления счетов;
- Поддержка
- маркетинговых сообщений;
- рекомендаций по лучшим методикам на основе использования Azure.

Введите адрес электронной почты, на который вы хотите получать сообщения об учетной записи. Указывая адрес электронной почты, вы соглашаетесь на получение сообщений от корпорации Майкрософт.

![Пример запроса на обновление контактных данных](./media/billing-how-to-change-azure-account-profile/update-contact-information.png)

### <a name="change-your-contact-email-address"></a>Изменение контактного адреса электронной почты

Вы можете изменить свой контактный адрес электронной почты с помощью одного из приведенных ниже способов.

* Войдите в [Центр учетных записей Azure](https://account.azure.com/subscriptions). Выберите **Изменить сведения**, а затем обновите адрес электронной почты профиля. При обновлении адреса электронной почты в профиле адрес электронной почты для входа не обновляется.

* На портале Azure перейдите в область [Контактные данные](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade) и введите адрес электронной почты. На адрес, который вы предоставляете, приходят служебные и маркетинговые сообщения. Адрес электронной почты в профиле Центра управления учетной записью Azure автоматически обновляется для соответствия. При обновлении адреса электронной почты в профиле адрес электронной почты для входа не обновляется.

* На [портале Azure](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade) выберите значок со своими инициалами или изображением. Затем выберите контекстное меню ( **...** ). Далее в меню выберите **Мои контактные данные** и введите адрес электронной почты. На адрес, который вы предоставляете, приходят служебные и маркетинговые сообщения. Адрес электронной почты в профиле Центра управления учетной записью Azure автоматически обновляется для соответствия. При обновлении адреса электронной почты в профиле адрес электронной почты для входа не обновляется.

![Пример обновления адреса электронной почты в Azure](./media/billing-how-to-change-azure-account-profile/azure-contact-information.png)

### <a name="opt-out-of-marketing-emails"></a>Отказ от маркетинговых сообщений

Чтобы отказаться от получения маркетинговых сообщений, выполните приведенные ниже действия.

1. Перейдите к [форме запроса](https://account.microsoft.com/profile/permissions-link-request), чтобы отправить запрос, используя адрес электронной почты профиля. Вы получите ссылку по электронной почте, чтобы обновить параметры.
2. Щелкните ссылку, чтобы открыть страницу **Manage communication permissions** (Управление разрешениями связи). На этой странице отображаются типы маркетинговых сообщений, которые выбраны для этого адреса электронной почты. Очистите все разделы, от которых хотите отказаться, а затем выберите **Сохранить**.

![Пример страницы для управления разрешениями связи](./media/billing-how-to-change-azure-account-profile/manage-communication-permissions.png)

Когда вы отказываетесь от маркетинговых сообщений, вы по-прежнему получаете уведомления службы, основанные на учетной записи.

## <a name="change-the-subscription-name"></a>Изменение имени подписки

1. Войдите на [портал Azure](https://portal.azure.com), выберите **Подписка** в левой области, а затем выберите подписку, которую вы хотите переименовать.
1. Выберите **Обзор**, а затем на панели команд щелкните **Переименовать**.
    ![Пример переименования подписки Azure](./media/billing-how-to-change-azure-account-profile/rename-sub.png)
1. После изменения имени выберите **Сохранить**.

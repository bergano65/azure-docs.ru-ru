# <a name="additional-documentation"></a>Дополнительная документация 

Некоторые ресурсы, находящиеся за пределами этого веб-сайта docs.microsoft.com, могут помочь вам понять, как работать с гибридным кэшем Microsoft Azure ФКСТ пограничной фильтр.

## <a name="avere-vfxt-for-azure-cache-documentation"></a>Документация по авере Вфкст для кэша Azure

[Авере вфкст для Azure](https://docs.microsoft.com/azure/avere-vfxt/) — это аналогичный продукт кэша кластера, который использует виртуальные узлы в Microsoft Azure. Он создается с портал Azure, но использует то же программное обеспечение для настройки панели управления, что и кэш оборудования Azure ФКСТ погранично фильтр. 

## <a name="legacy-hardware-documentation"></a>Документация по устаревшему оборудованию 

Эта технология ранее использовалась в продуктах от Авере Systems. Документация для пользователей этих старых продуктов доступна на [веб-сайте документации Авере Legacy](https://azure.github.io/Avere/). 

В частности, эти документы могут иметь полезные сведения:

* [Руководство по настройке кластера](https://azure.github.io/Avere/legacy/ops_guide/4_7/html/ops_conf_index.html) . включает общие сведения об общих задачах по настройке и управлению кластером, а также подробные объяснения страниц параметров кластера. 

* [Панель мониторинга](https://azure.github.io/Avere/legacy/dashboard/4_7/html/ops_dashboard_index.html) — сведения о возможностях вкладки панели управления панель **мониторинга**

* [Путеводитель по](https://azure.github.io/Avere/legacy/create_cluster/4_8/html/create_index.html) созданию кластера фкст. Обзор создания кластера из предыдущих продуктов

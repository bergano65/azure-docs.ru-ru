# <a name="upgrade-your-azure-free-account-or-azure-for-students-starter-account"></a>Обновление бесплатной учетной записи Azure или учетной записи Начального набора Microsoft Azure для учащихся

[Бесплатную учетную запись Azure](https://azure.microsoft.com/free/) или [учетную запись начального набора Microsoft Azure для учащихся](https://azure.microsoft.com/offers/ms-azr-0144p/) можно обновить до [оплаты по мере использования](https://azure.microsoft.com/offers/ms-azr-0003p/) на портале Azure.

Если вы зарегистрировались с помощью [учетной записи Начального набора Microsoft Azure для учащихся](https://azure.microsoft.com/offers/ms-azr-0144p/) и имеете право на [бесплатную учетную запись Azure](https://azure.microsoft.com/free/), вы можете перейти к бесплатной учетной записи. При обновлении вы получите 200 долларов кредитов Azure и 12 месяцев бесплатных служб.

<a id="freetrial"></a>

## <a name="upgrade-your-azure-free-account"></a>Обновление бесплатной учетной записи Azure

При обновлении бесплатной учетной записи Azure вы сохраняете оставшийся кредит в течение полных 30 дней с момента регистрации. Пример: Если вы зарегистрировались 1 ноября и выполнили обновление 5 ноября, все неиспользованные деньги на счете будут доступны до 30 ноября. Кроме того, вам предоставляется доступ к бесплатным службам в течение 12 месяцев.

1. Войдите на [портал Azure](https://portal.azure.com)
1. Выполните поиск по запросу **Подписки**.

    ![Снимок экрана, на котором показан поиск](./media/upgrade-azure-subscription/search-subscriptions-ibiza.png)

1. Выберите подписку, созданную при регистрации бесплатной учетной записи Azure.
1. В разделе "Обзор" подписки щелкните кнопку **Обновить подписку** на панели команд. Если кнопка "Обновить подписку" не отображается, щелкните баннер обновления в верхней части страницы.

    ![Снимок экрана, на котором изображена кнопка "Обновить"](./media/upgrade-azure-subscription/free-upgrade-button.png)

1. Если у вас еще нет метода оплаты для подписки, вам будет предложено добавить его.

1. Для проверки личности может потребоваться ввести номер телефона.

1. Введите имя подписки.

     ![Снимок экрана, на котором показано имя](./media/upgrade-azure-subscription/free-upgrade-name.png)

1. Выберите план поддержки для своей подписки. Дополнительные сведения о планах поддержки см. в статье [Планы поддержки Azure.](https://azure.microsoft.com/us/support/plans/)

1. Щелкните **Обновить**.

## <a name="upgrade-your-azure-for-students-starter-account"></a>Обновление учетной записи Начального набора Microsoft Azure для учащихся

### <a name="upgrade-to-an-azure-free-account"></a>Обновление до бесплатной учетной записи Azure

Если вы имеете право, выполните приведенные ниже действия, чтобы выполнить обновление до бесплатной учетной записи Azure.

1. Войдите на [портал Azure](https://portal.azure.com)
1. Выполните поиск по запросу **Подписки**.

    ![Снимок экрана, на котором показан поиск](./media/upgrade-azure-subscription/search-subscriptions-ibiza.png)

1. Выберите подписку, которая была создана при регистрации учетной записи Начального набора Microsoft Azure для учащихся.
1. В разделе "Обзор" подписки щелкните **Обновить подписку** на панели команд.

    ![Снимок экрана, показывающий кнопку "Обновить" для учащихся](./media/upgrade-azure-subscription/student-upgrade-ibiza.png)

### <a name="upgrade-to-pay-as-you-go-rates"></a>Обновление до подписки с оплатой по мере использования

1. Если вы выполняете обновление до тарифов с оплатой по мере использования и еще не указали метод оплаты подписки, вам будет предложено добавить его.
1. Для проверки личности может потребоваться ввести номер телефона.
1. Введите имя подписки.
1. Выберите план поддержки для своей подписки. Дополнительные сведения о планах поддержки см. в статье [Планы поддержки Azure.](https://azure.microsoft.com/us/support/plans/)
1. Щелкните **Обновить**.

## <a name="next-steps"></a>Дальнейшие действия

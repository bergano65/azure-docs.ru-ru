# <a name="pool-autoscale-event"></a>Событие автомасштабирования пула

 Это событие создается после выполнения автоматического масштабирования пула. Содержимое журнала предоставит формулу автомасштабирования и результаты оценки для пула.

 В следующем примере показан текст события автомасштабирования пула для автоматического масштабирования пула, который завершился сбоем из-за недостаточного примера данных.

```
{
    "id": "myPool1",
    "timestamp": "2020-09-21T18:59:56.204Z",
    "formula": "...",
    "results": "...",
    "error": {
        "code": "InsufficientSampleData",
        "message": "Autoscale evaluation failed due to insufficient sample data",
        "values": [{
                "name": "Message",
                "value": "Line 15, Col 44: Insufficient data from data set: $RunningTasks wanted 70%, received 50%"
            }
        ]
    }
}
```

|Элемент|Тип|Примечания|
|-------------|----------|-----------|
|`id`|Строка|Идентификатор пула.|
|`timestamp`|Дата и время|Метка времени выполнения автоматического масштабирования.|
|`formula`|Строка|Формула, определенная для автоматического масштабирования.|
|`results`|Строка|Результаты вычисления для всех переменных, используемых в формуле.|
|[`error`](#error)|Сложный тип|Подробные сведения об ошибке автоматического масштабирования.|

###  <a name="error"></a>Ошибок: <a name="error"></a>

|Имя элемента|Тип|Примечания|
|------------------|----------|-----------|
|`code`|Строка|Идентификатор для ошибки автоматического масштабирования. Коды являются инвариантными и предназначены для использования программными средствами.|
|`message`|Строка|Сообщение, описывающее ошибку автоматического масштабирования, предназначенное для вывода в пользовательском интерфейсе.|
|`values`|Массив|Список пар "имя-значение", описывающих дополнительные сведения об ошибке автоматического масштабирования.|

## <a name="create-a-key-vault"></a>Создание хранилища ключей

1. На портале Azure на начальном экране выберите **Создать ресурс**.
1. Введите *Key vault* в поле поиска **Marketplace** и выберите **Key Vault** в результатах поиска.
1. Нажмите кнопку **создания**.  Откроется экран "Создать хранилище ключей".
1. Выберите нужную **группу ресурсов** или создайте новую.
1. Присвойте Key Vault имя, введя его в поле **Key Vault**.
1. В этом примере вы оставите для **параметров восстановления** значения по умолчанию.
1. По завершении выберите **Next: политика доступа >** . Появится экран "Политика доступа".
1. Предоставьте пользователю список необходимых разрешений для хранилища ключей.  Разрешений по умолчанию должно быть достаточно.
1. Нажмите **Далее: сеть**. Откроется экран "Сеть".
1. Выберите тип конечной точки, которую вы хотите использовать:
1. Щелкните **Review + create** (Просмотреть и создать).
